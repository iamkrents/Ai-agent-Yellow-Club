# Роли и зоны ответственности

## Структура

CEO / Основатель

Операционный менеджер
- Преподаватели по операционным вопросам: расписание, замены, классы, системы.

Ведущий клиент-менеджер
- Клиент-менеджер.

Старший преподаватель / методист
- Преподаватели по методике обучения, качеству занятий, развитию преподавателей.

Маркетинг
- Маркетолог.
- SMM.

Бухгалтер.

## К кому обращаться

### Операционный менеджер
Обращаться, если:
- нужно заменить преподавателя;
- вопрос по расписанию;
- проблема в учебном классе или оборудовании;
- не работает CRM, система или платформа;
- непонятно, как организован процесс.

### Клиент-менеджеры
Обращаться, если:
- родитель задаёт вопрос или недоволен;
- ученик пропускает занятия;
- нужно связаться с клиентом;
- вопрос по заявкам.

### Старший преподаватель / методист
Обращаться, если:
- преподаватель не понимает, как провести занятие;
- есть вопросы по программе или уроку;
- нужно улучшить занятие;
- нужно подобрать уровень программы под группу.

### Маркетолог / SMM
Обращаться, если:
- есть идеи по рекламе;
- есть идеи по контенту;
- нужно передать отзыв или материал для публикации.

### Бухгалтер
Обращаться, если:
- вопросы по оплатам;
- вопросы по зарплате;
- вопросы по документам.

## Принцип взаимодействия

- сначала обращаться к своему прямому руководителю;
- если вопрос не решился - поднимать выше по структуре;
- в срочных ситуациях - писать напрямую ответственному;
- не оставлять проблему без реакции;
- чем раньше сообщили, тем проще решить.
