	**Ожидаемый результат урока**
	```kotlin
// Логические операторы:
// && - и
// || - или
// ! - не

// Обратить внимание на это ->
// Если у нас несколько логических выражений, их нужно разделять скобками, иначе будет ошибка

// && (and)
println((5 > 4) && (6 < 7)) // true
println((5 > 4) && (6 < 5)) // false
println((5 < 4) && (6 < 5)) // false

// || (or)
println((5 > 4) || (6 < 7)) // true
println((5 > 4) || (6 < 5)) // true
println((5 < 4) || (6 < 5)) // false

// Пример с походом в магазин:
// У вас есть кола и чипсы?
// У вас есть кола или фанта?

// ! (not) - переворачивает значение одного условия
println((5 > 4) && (6 > 7))  // результат False
println((5 > 4) && !(6 > 7)) // результат True

// 4 способами сделайте результат True
println((5 > 4) && (6 > 7))
// 4 способа сделать True:
println((5 > 4) && (6 < 7))
println((5 > 4) || (6 > 7))
println((5 > 4) || (6 < 7))
println((5 > 4) && !(6 > 7))

// Оператор принадлежности in
// Обратим внимание, что проверка принадлежности буквы к слову можно писать как в одинарных кавычках, так и в двойных
// так как в Kotlin есть отдельный тип данных Char, который может работать с String
println("а" in "привет")
println('и' in "привет")

//Так же мы можем проверять, есть ли какое-то число в некой последовательности
println(1 in 1..6)// true
println(2 !in 10..25) // true
	```

```kotlin
// Задание 1
// В школе проходит олимпиада для 6 - 9 классов. Необходимо сделать алгоритм, который:
// - запросит ваше имя
// - запросит вашу фамилию
// - запросит ваш класс без буквы
// - Выдаст сообщение: "Имя Фамилия, проверьте, можете ли вы принимать участие в олимпиаде: "
// - Выдаст результат True или False

println("Регистрация на олимпиаду.")
print("Введите ваше имя: ")
val name = readln()
print("Введите вашу фамилию: ")
val surname = readln()
println("Введите номер вашего класса: ")
val grade = readln().toInt()
println("$name $surname, проверьте, можете ли вы принимать участие в олимпиаде: ")
//здесь есть 2 варианта проверки 
println((grade >= 6) && (grade <= 9))
//либо
println(grade in 6..9)

// Задание 2
// Есть аквапарк с горкой, на которую пускают, если:
// - возраст больше 10 лет
// - или рост больше 150 см
// Нужно написать программу, которая:
// - приветствует посетителя по имени
// - запрашивает возраст и рост
// - проверяет возраст и рост на допустимые значения
// - выдает решение в виде допуска на горку в виде: True или False

print("Добро пожаловать в аквапарк!\nВведите ваше имя:")
val name = readln()
println("Добро пожаловать в аквапарк, $name!")
print("Введите ваш возраст: ")
val age = readln().toInt()
print("Введите ваш рост: ")
val height = readln().toInt()
print("Проверьте, допущены ли вы к нашей самой страшной горке, $name :")
//здесь проверка идет такая: или возраст от 10 или просто от 150, поэтому просто проверяем оператором || (или)
println((age >= 10) || (height >= 150))
```
