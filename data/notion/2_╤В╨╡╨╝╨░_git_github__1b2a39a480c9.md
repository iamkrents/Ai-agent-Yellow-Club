# Локальный репозиторий
## Установка Git
- проверка работы Git c помощью корневой команды
```bash
git
```
<callout icon="💡" color="gray_bg">
	Необходимо скачать и установить Git с официального сайта
	[Git](https://git-scm.com/)
</callout>
## Инициализация репозитория
- инициализация репозитория
```bash
git init
```
- проверка статуса
```bash
git status
```
## Добавление в индекс
- добавление их в индекс (промежуточный этап)
```bash
git add
```
## .gitignore
- создать файл .gitignore и поместить туда название файла, который мы хотим игнорировать
```python
.idea
venv
```
- проверка статуса
```bash
git status
```
## Коммиты
- сохранение в локальный репозиторий
```bash
git commit -m 'Комментарий к коммиту'
```
## Автор коммитов
- указать автора репозитория/коммита/проекта
```bash
  git config --global user.email "you@example.com"
  git config --global user.name "Your Name"
```
<callout icon="💡" color="gray_bg">
	Если выдает ошибку с правами доступа, то можно запустить терминал от имени администратора и  выполнить настройку там.
</callout>
## История коммитов
- посмотреть историю коммитов
```bash
git log
```

# Удаленный репозиторий
## Создание репозитория
- создать репозиторий на Github в аккаунте yellowclub.by
## Синхронизация локального и удаленного репозитория
- связать локальный и удаленный репозитории
	- выбрать https формат подключения
	- разобрать нижнее окно с кодом
		- git remote - создание связи
		- переименование главной ветки - разобрать команды и зачем это нужно
		```bash
git branch
		```
		- git push - отправка в удаленный репозиторий
	- скопировать код и вставить в терминал в PyCharm
- проверить репозиторий Github на наличие файлов
