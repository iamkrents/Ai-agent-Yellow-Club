# Операционные системы
- разобрать что такое ОС и какие бывают ОС (MacOS, Windows, Linux)

- разобрать как мы взаимодействуем с ядром
# **Терминал**
- разобрать, что такое терминал/консоль
	- создать папку и назвать своим ФИО (на англ)
	- открыть терминал из папки и разобрать подробно, что мы в нем видим
	- закрыть проводник и через графический интерфейс дойти до папки 
	- рассказать про файловую систему и адреса 
	- вернуться в терминал, сравнить адрес с тем, что мы видим в графическом интерфейсе
- научиться базовым командам для работы через терминал
	- закрыть терминал и открыть его из корневой папки (Диск C)
	- пройти базовые команды и с помощью них добраться до нашей папки проекта
		- отображение содержимого директории (directory)
			```bash
dir 
			```
		- изменение текущей директории (change directory)
			```bash
cd <путь_к_папке>
			```
		- переход на уровень выше в иерархии директорий
			```bash
cd ../
			```
	1. Дополнительно можно разобрать команды
		- создание новой папки (make directory)
			```bash
mkdir <имя>
			```
			- rmdir
		- удаление папки (remove directory)
			```bash
rmdir <имя>
			```
# **Окружение**
- разобрать понятие виртуального окружения
- создать окуружение через консоль в папке
```bash
python -m venv venv
```
Нажать именно на папку и открыть с помощью PyCharm.
Активировать виртуальное окружение.
```bash
venv\Scripts\activate
```

# Фреймворк Aiogram
- рассказать, что понадобиться для создания telegram-бота
- посмотреть установленные библиотеки
	- зайти в терминал PyCharm
	- использовать команду
		```bash
pip list
		```
- установить фреймворк Aiogram
	- зайти на сайт [**PYPI.ORG**](http://PYPI.ORG)** **и найти aiogram
	- скопирвать команду для установки
	- вставить в терминал и запустить (нажать Enter)
	- проверить установку с помощью команды `pip list`
