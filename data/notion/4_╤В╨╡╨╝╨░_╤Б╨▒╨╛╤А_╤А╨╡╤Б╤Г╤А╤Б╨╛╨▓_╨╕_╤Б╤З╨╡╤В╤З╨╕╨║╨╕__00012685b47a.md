# **Создание ресурса**
## Отрисовка ресурса
- нарисовать объект-ресурс, соответствующий стилю и плану игры
- назвать ресурс понятным именем
## Анимация ресурса
- создать дополнительный кадр для анимации
- включить зацикливание анимации
- настроить скорость смены кадров
# **Настройка визуального счётчика**
## **Создание слоя интерфейса**
- перейти во вкладку Слои
- создать новый слой, назвать его UI или HUD
- убедиться, что этой слой расположен выше остальных
## **Добавление иконки ресурса в интерфейс**
- продублировать изображение ресурса, чтобы создать иконку счетчика
- переименовать ресурс, чтобы не возникало путаницы
- разместить иконку в верхнем углу экрана
- перенести иконку на слой UI
## **Создание числового счётчика**
- добавить текстовый объект
- ввести значение по умолчанию - 0
- настроить шрифт, цвет и размер текста
- разместить число рядом с иконкой ресурса
- перенести текст на слой UI
# **Настройка логики счётчика**
## **Добавление поведения**
- добавить к тексту поведение Счётчик
- настроить свойства поведения
## **Программирование работы счетчика**
- добавить в событие сбора ресурса действие увеличения значения счетчика
- запустить игру и убедиться, что значение счётчика меняется при сборе ресурса
# **Структурирование событий**
## **Создание групп событий**
- открыть редактор событий
- создать 3 группы:
	- анимация персонажа
	- взаимодействие с объектами
	- враги и препятствия
- переместить соответствующие события в нужные группы
## **Добавление комментариев к событиям**
- добавить комментарии внутри групп
- переместить соответствующие события к нужным комментариям
