/*
	Небольшое вступление, мы забыли про операцию деления с остатком, она не сложная, поэтому сразу закрепим на практике.
	Знак деления с остатком - %. Пример 3 % 2 = 1, то есть остаток.
*/
println(3 % 2)
println(4 % 2)
/*
	Четное или нечетное. Ввести с клавиатуры одно целое число. Проверить является ли число четным или нечетным, результат вывести на экран.
	Решение: Мы должны понять, что четным число считается, когда отстаток от деления на 2 равен 0, если остаток отличается, то число
	считается нечетным.
*/
print("Введите число: ")
val num = readln().toInt()
if(num % 2 == 0){
	println("Четное")
}else{
	println("Нечетное")
}

/*
	Задача 1. Арифметические операции. Палиндром. С клавиатуры вводится число до 99. Нужно проверить, является ли оно палиндромом. Палиндром - 
	это слово, фраза, число, которые читаются слева направо и справа налево одинаково. Пример: привет - не палиндром, шалаш - палиндром,
	99 - палиндром
*/

print("Введите двузначное число: ")
val num = readln().toInt()
// Если остаток от деления равено результату деления на 10, то число палиндром 88 / 10 = 8 и 88 % 10 = 8
if((num / 10) == (num % 10)){
	println("$num - палиндром")
}else{
	println("$num - не палиндром")
}

/*
	Задача 2. Операции сравнения и логические операции. Программа просит ввести твой рост, потом рост твоего соседа. В результате
	на экране должен появиться ответ, выше ли ты своего соседа. (true, false)
*/

print("Введи свой рост: ")
val myHeight = readln().toInt()
print("Теперь введи рост твоего соседа: ")
val neighborHeight = readln().toInt()
println("Я выше? - ${myHeight > neighborHeight}")

/*
	Задача 3. Условная конструкция if-elif-else. Программа просит ввести одного игрока камень, ножницы или бумагу, после просит то же самое
	второго игрока. Исходя из того, что ответили игроки, на экране должно появиться сообщение о том, что игрок1 или игрок2 победил.
	При желании, можно добавить возможность игрокам ввести их имена. Возможен вариант с многочисленными else-if, это не ошибка.
*/

print("Игрок 1, введите свое имя: ")
val firstName = readln()
print("Игрок 2, введите свое имя: ")
val secondName = readln()
print("$firstName, выберите камень-ножницы-бумага и введите свой выбор: ")
val firstChoice = readln().lowercase()
print("$secondName, выберите камень-ножницы-бумага и введите свой выбор: ")
val secondChoice = readln().lowercase()

if ((firstChoice == "камень" && secondChoice == "ножницы")
	|| (firstChoice == "ножницы" && secondChoice == "бумага")
	|| (firstChoice == "бумага" && secondChoice == "камень")
) {
	println("$firstName победил!")
} else if ((secondChoice == "камень" && firstChoice == "ножницы")
	|| (secondChoice == "ножницы" && firstChoice == "бумага")
	|| (secondChoice == "бумага" && firstChoice == "камень")
) {
	println("$secondName победил!")
} else if ((secondChoice == "камень" && firstChoice == "камень")
	|| (secondChoice == "ножницы" && firstChoice == "ножницы")
	|| (secondChoice == "бумага" && firstChoice == "бумага")
) {
	println("Ничья!")
} else {
	println("Кто-то из игроков неверно ввел свой выбор! Начните заново...")
}

/*
	Задача 4. Условная конструкция when. Введи день недели и исходя из него, на экране появится информация о количестве уроков в этот
	день. День недели может быть введен как цифрой, так и словом. |Если начнут делать через if-else, попросить вспомнить, может быть
	есть что-то лучше, когда выбор большой|
*/

print("Введите день недели: ")
	val dayOfWeek = readln()
	when(dayOfWeek.lowercase()){
		"понедельник" -> { println("У тебя 3 урока") }
	  "вторник" -> { println("У тебя 4 урока") }
	  "среда" -> { println("У тебя 2 урока") }
	  "четверг" -> { println("У тебя 1 урок") }
	  "пятница" -> { println("У тебя 6 уроков") }
	  "суббота" -> { println("У тебя 2 факультатива") }
	  "воскресенье" -> { println("Отдыхай!") }
	else -> { println("Такого дня нет...") }
  }
  // Вариант с lowercase() имеет место быть, так как ввод может быть разных регистров.

/*
	Задача 5. Списки. Сумма элементов списка. Создан массив с n-ым количеством целочисленных элементов. Задача сложить каждый элемент и вывести результат 
	на экран. Так как циклы мы еще не знаем, сумма будет вычисляться путем индексов массива и переменных.
*/
	// самое быстрое решение
val arr = listOf(1,2,4,5)
println(arr[0] + arr[1] + arr[2] + arr[3])
/*
	Возможный вариант решения:
	val arr = listOf(1,2,4,5)
	val el1 = arr[0]
	val el2 = arr[1]
	val el3 = arr[2]
	val el4 = arr[3]
	val res = el1 + el2 + el3 + el4 
	println(res)
*/

/*
	Задача 6. Списки и методы списков. Список дел. Введи через запятую с пробелом список дел на сегодня. После чего,
	удали дело, которое ты уже сделал по индексу и еще одно по названию, а потом добавь одно, которое хотел бы успеть сделать. На экран вывести начальный список, список после 
	удаления сделанных дел, список после добавления дел. |mutableList|
*/

print("Введи список дел через запятую с пробелом: ")
val list = readln().split(", ").toMutableList()
println(list)
print("Введи название дела, которое ты уже выполнил: ")
val actionToRemove = readln()
list.remove(actionToRemove)
println(list)
print("Введи индекс дела, которое ты уже выполнил: ")
val actionToRemoveIndex = readln().toInt()
list.removeAt(actionToRemoveIndex)
println(list)
print("Введи название дела, которое ты хочешь выполнить: ")
val actionToAdd = readln()
list.add(actionToAdd)
println(list)
	
/*
	Задача 7. Создание строки из последнего символа элементов массива. С клавиатуры через пробел вводится 4 любых слова,
	у каждого слова взять первую букву и соеденить их. Сделать проверку на размер списка. (Размер списка должен быть 4,
	если же он будет меньше 4, то появится ошибка)
	*/
print("Введите 4 слова через пробел: ")
val strList = readln().split(" ")
val firstLetter:Char
val secondLetter:Char
val thirdLetter:Char
val fourthLetter:Char
if(strList.size == 4){
	firstLetter = strList[0].first()
	secondLetter = strList[1].first()
	thirdLetter = strList[2].first()
	fourthLetter = strList[3].first()
	val result = "$firstLetter$secondLetter$thirdLetter$fourthLetter"
	println(result)
}else{
	println("Введите слова заново")
}
