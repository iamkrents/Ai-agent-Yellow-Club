# Регистрация аккаунта
- зарегистрировать аккаунт аккаунт в Figma
- подтвердить аккаунт через почту
- выдать доступ к шаблону
# Интерфейс и навигация в Figma
- изучить рабочую область
- рассмотреть панель слои
- рассмотреть панель инструментов
- рассмотреть панель свойств
- научиться навигации
	- изучить инструмент “рука”
	- изучить сочетание клавиш `пробел`+`ЛКМ`
# Текст
- изменить имя и фамилию
## Цвет
- разобрать палитру цветов
- изменить цвет текста
## Размер
- разобрать единицу измерения - px
- изменить размер текста
## Шрифт
- изменить шрифт
- разобрать дополнительные настройки шрифта (опционально)
# Фон
## Заливка
- изменить цвет фрейма
## Градиент
- переключиться в режим градиента
- разобрать свойства градиента
	- цвет
	- угол
	- масштаб
	- тип
- настроить градиент
# Кнопка
## Фон
- изменить заливку фона кнопки сплошным цветом
- настроить градиент фона кнопки
## Обводка
- разобрать свойства обводки
	- толщина
	- цвет
	- тип
- изменить значения свойств
## Текст
- изменить значения свойств текста
	- цвет
	- размер
	- шрифт
## Настройки при наведении (опционально)
- изменить значения свойств текста при наведении
	- цвет
	- размер
	- шрифт
# Изображение
- выбрать и сохранить изображение на рабочий стол
- загрузить изображение в элипс
- кадрировать изображение (опционально)
# Стилизация
- выбрать 2-3 цвета, которые есть на изображении
- стилизовать пройденные элементы исходя из выбранных цветов
	- текст
	- фон
	- кнопка
# Повторение
- в каком сервисе работали?
- как мы может перемещаться по рабочему пространству?
- какие объекты разобрали?
- какие свойства текста научились менять?
- что такое градиент?
- какие свойства есть у градиента?
- какие свойства меняли у кнопки?
- какие свойства есть у обводки?
- как мы пришли к единому стилю работы?
- что больше всего запомнилось?
