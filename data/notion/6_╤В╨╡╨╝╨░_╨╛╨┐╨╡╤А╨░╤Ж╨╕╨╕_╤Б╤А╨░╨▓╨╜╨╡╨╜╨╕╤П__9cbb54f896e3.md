# Операции сравнения в языке С
<callout icon="🧑‍🎓" color="gray_bg">
	Для чего нужны операции сравнения?
	<callout icon="🙋‍♂️" color="yellow_bg">
		Чтобы сравнить какие-либо значения.
	</callout>
</callout>
Операции сравнения позволяют сравнить два значения и возвращают 1, если выражение истинно, и 0, если выражение ложно.
## Операторы сравнения
<callout icon="🧑‍🎓" color="gray_bg">
	Какие операторы сравнения вы знаете?
	<callout icon="🙋‍♂️" color="yellow_bg">
		Больше, меньше, больше либо равно, меньше либо равно, равно.
	</callout>
</callout>
Операторы сравнения:
<span color="blue">**\>**</span> — больше, чем
<span color="blue">**\<**</span> — меньше, чем
<span color="blue">**\>=**</span> — больше или равно
<span color="blue">**\<=**</span> — меньше или равно
<span color="blue">**==**</span> — равно
<span color="blue">**!=**</span> — не равно
## Виды операций сравнения
Синтаксис операций сравнения выглядит так:
```c
операнд_1 оператор_сравнения операнд_2
```
<span color="blue">**Операнд**</span> - это элемент (или выражение), который участвует в операции. Операнды могут быть числами, переменными, константами, функциями или выражениями.
<span color="blue">**Оператор сравнения**</span> - это символ, который указывает какую операцию необходимо произвести.
Для примера объявим переменные <span color="blue">**a**</span> и <span color="blue">**b**</span>, которым присвоим целочисленные значения и проведем с ними различные операции сравнения.
```c
#include <stdio.h>

int main()
{
	int a = 11;
	int b = 3;

	return 0;
}
```
### **Больше, чем**
Возвращает 1, если первый операнд больше второго, и 0, если первый операнд меньше или равен второму.
```c
#include <stdio.h>

int main()
{
	int a = 11;
	int b = 3;
	
	a > b;
	printf("%d \n", a > b);

	return 0;
}
```
Результатом будет вывод в консоль 1, так как 11 больше 3.
### **Меньше, чем**
Возвращает 1, если первый операнд меньше второго, и 0, если первый операнд больше или равен второму.
```c
#include <stdio.h>

int main()
{
	int a = 11;
	int b = 3;
	
	a < b;
	printf("%d \n", a < b);

	return 0;
}
```
Результатом будет вывод в консоль 0, так как 111 больше 3.
### **Больше или равно**
Возвращает 1, если первый операнд больше или равен второму, и 0, если первый операнд меньше второго.
```c
#include <stdio.h>

int main()
{
	int a = 11;
	int b = 3;
	
	a >= b;
	printf("%d \n", a >= b);

	return 0;
}
```
Результатом будет вывод в консоль 1, так как 11 больше 3.
### **Меньше или равно**
Возвращает 1, если первый операнд меньше или равен второму, и 0, если первый операнд больше второго.
```c
#include <stdio.h>

int main()
{
	int a = 11;
	int b = 3;
	
	a <= b;
	printf("%d \n", a <= b);

	return 0;
}
```
Результатом будет вывод в консоль 0, так как 11 больше 3.
### **Равно**
Возвращает 1, если оба выражения равны, и 0, если они не равны:
```c
#include <stdio.h>

int main()
{
	int a = 11;
	int b = 3;
	
	a == b;
	printf("%d \n", a == b);

	return 0;
}
```
Результатом будет вывод в консоль 0, так как числа 11 и 3 не равны.
<callout icon="💡" color="gray_bg">
	Один знак “<span color="blue">**=**</span>” - это присвоение значения переменной, два знака “<span color="blue">**==**</span>” - это операция сравнения.
</callout>
### **Не равно**
Возвращает 1, если первый операнд не равен второму, и 0, если оба операнда равны:
```c
#include <stdio.h>

int main()
{
	int a = 11;
	int b = 3;
	
	a != b;
	printf("%d \n", a != b);

	return 0;
}
```
Результатом будет вывод в консоль 1, так как числа 11 и 3 не равны.
