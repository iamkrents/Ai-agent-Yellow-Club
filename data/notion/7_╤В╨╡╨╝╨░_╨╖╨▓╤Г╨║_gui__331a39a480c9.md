<summary>1. Раздел **Data (Данные)**</summary>
	<callout icon="💡" color="yellow_bg">
		### Archivable
		Определяет, можно ли **скопировать объект** при клонировании.
		- ✅ включено — звук будет копироваться
		- ❌ выключено — не будет копироваться
		Используется редко.
	</callout>
	<callout icon="💡" color="yellow_bg">
		### ClassName
		Тип объекта.
		В данном случае:
		**Sound** — объект, который воспроизводит аудио.
		Изменить нельзя.
	</callout>
	<callout icon="💡" color="yellow_bg">
		### Name
		Имя объекта.
		Например:
		```plain text
BackgroundMusic
DoorSound
ExplosionSound
		```
		Используется в скриптах:
		```plain text
script.Parent.BackgroundMusic:Play()
		```
	</callout>
	<callout icon="💡" color="yellow_bg">
		### Parent
		Показывает **где находится объект**.
		В вашем случае:
		```plain text
Workspace
		```
		Значит звук расположен в мире игры.
	</callout>
	<callout icon="💡" color="yellow_bg">
		### UniqueId
		Уникальный ID объекта внутри проекта.
		Используется системой Roblox.
		Менять нельзя.
	</callout>

## Делаем скример
Создаем парт
Добавляем звук
Добавляем ScreenGui
	Внутрь его добавляем ImageLabel
Обязательно настраиваем у картинки
	**Size** → `{1, 0},{1, 0}`
	**Position** → `{0, 0},{0, 0}`
	**Image** → вставить ID картинки
	**BackgroundTransparency** → `1`
У ScreenGui ставим **Enabled **= false, чтобы картинки не было видно изначально и **IgnoreGuiInset **= true чтобы не было видно верхнюю панель
Добавляем скрипт
```lua
-- Получаем кубик (Part), к которому привязан скрипт
local part = script.Parent

-- Находим звук внутри кубика (это шаблон, будем его копировать)
local sound = part:WaitForChild("Sound")

-- Находим GUI (картинку скримера)
local gui = part:WaitForChild("ScreenGui")

-- Переменная-защита от многократного срабатывания
local debounce = false

-- Событие: когда кто-то касается кубика
part.Touched:Connect(function(hit)

	-- Пытаемся определить игрока (если это персонаж)
	local player = game.Players:GetPlayerFromCharacter(hit.Parent)

	-- Если это не игрок ИЛИ уже срабатывало → выходим
	if not player or debounce then return end

	-- Включаем "защиту от спама"
	debounce = true

	-- Клонируем GUI (создаём копию картинки)
	local cloneGui = gui:Clone()

	-- Помещаем GUI в PlayerGui (экран игрока)
	cloneGui.Parent = player.PlayerGui

	-- Включаем GUI, чтобы он стал видимым
	cloneGui.Enabled = true

	-- КЛОНИРУЕМ звук
	local soundClone = sound:Clone()

	-- Кладём звук в PlayerGui (чтобы он играл только у этого игрока)
	soundClone.Parent = player.PlayerGui

	-- Запускаем звук
	soundClone:Play()

	-- Ждём 3 секунды (длительность скримера)
	task.wait(3)

	-- Удаляем картинку
	cloneGui:Destroy()

	-- Останавливаем звук (на всякий случай)
	soundClone:Stop()

	-- Удаляем звук (чтобы не играл дальше)
	soundClone:Destroy()

	-- Небольшая пауза перед повторным срабатыванием
	task.wait(2)

	-- Снова разрешаем срабатывание
	debounce = false
end)
```
# Объяснение (для детей)
- **Touched** → срабатывает, когда кто-то коснулся кубика
- Мы проверяем: это игрок или нет
- **Clone()** → копируем GUI игроку
- **PlayerGui** → экран игрока
- **sound:Play()** → проигрываем звук
- **wait(3)** → ждём 3 секунды
- потом всё убираем
