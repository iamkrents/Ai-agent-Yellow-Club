# Материалы
<callout icon="☑️" color="gray_bg">
	Расскажите, что такое материалы.
</callout>
Материалы регулируют внешний вид мешей и других объектов. Они определяют вещество, из которого сделан объект, его цвет и текстуру, а также то, как с ним будет взаимодействовать свет.
Материалы состоят из трёх компонентов:
- внешний вид поверхности
- объём внутри объекта
- смещение поверхности
На данный момент мы разберемся с первым компонентом, который определяет внешний вид поверхности.
# Назначение материалов

</callout>
- выбираем объект
- переходим во вкладку “Materials” в области “Properties”

## Назначение цвета и других свойств
<callout icon="☑️" color="gray_bg">
	Разберите и поменяйте базовые настройки материала кубика.
</callout>
- изменяем цвет (color)
- изменяем шероховатость (roughness)
- изменяем металлик (metallic)
<callout icon="💡" color="gray_bg">
	Не забудьте переключиться в режим просмотра “Material preview”, иначе материал не будет отображаться.
</callout>
## Работа со слотами
Материалы можно создать и затем назначить одному или нескольким объектам.
<callout icon="☑️" color="gray_bg">
	Создайте новый материал и примените его к текущему объекту.
</callout>
- добавляем новый слот к текущему кубику
- добавляем новый материал
- изменяем настройки нового материала, например, цвет
- удаляем старый слот с текущего объекта
## **Повторное использование материалов**
<callout icon="☑️" color="gray_bg">
	Добавьте уже созданный материал к новому объекту.
</callout>
- добавляем новый кубик
- добавляем новый слот
- выбираем созданный материал из списка
# Назначение материалов частям объекта
По умолчанию объекты имеют только один слот материалов, который назначает материал всему объекту. Если для разных частей меша нужны разные материалы, можно создать несколько слотов для материалов.
## Назначение разных материалов граням
<callout icon="☑️" color="gray_bg">
	Назначьте каждой грани кубика разные цвета.
</callout>
- создаем 6 материалов с разными цветами
- переходим в Edit Mode для выбора граней
- выбираем одну грань
- выбираем материал
- нажимаем кнопку “Asign” для назначения материала грани
- проделываем то же самое для других граней с другими материалами
## Создание новых граней

# Создание скина Стива
Давайте начнем делать скин Стиву. Можно реализовать как базовый скин из референса, так и свой скин.
## Голова
<callout icon="🧑‍🎓" color="gray_bg">
	Что необходимо будет сделать, чтобы мы могли назначать материалы для головы?
	<callout icon="🙋‍♂️" color="yellow_bg">
		Создать их!
	</callout>
</callout>
<callout icon="☑️" color="gray_bg">
	Добавьте все необходимые материалы для головы.
</callout>
- выбираем голову
- добавляем слот и новый материал
- переименовываем материал соответственно объекту (например, волосы)
- выбираем пипеткой соответствующий цвет
<callout icon="🧑‍🎓" color="gray_bg">
	Как теперь назначить эти материалы разным частям головы?
	<callout icon="🙋‍♂️" color="yellow_bg">
		Зайти в режим редактирования.
	</callout>
	А как создать новые плоскости для более мелких деталей?
	<callout icon="🙋‍♂️" color="yellow_bg">
		Использовать инструмент Loop Cut!
	</callout>
</callout>
<callout icon="☑️" color="gray_bg">
	Поделите кубик головы на нужное количество граней.
</callout>
- выбираем голову
- заходим в Edit Mode
- выбираем инструмент Loop Cut
- делим на нужное количество граней
<callout icon="☑️" color="gray_bg">
	Назначьте созданные материалы соответствующим граням.
</callout>
- выбираем грань
- выбираем материал
- нажимаем кнопку “Asign” для назначения материала грани
<callout icon="💡" color="gray_bg">
	Часть граней может быть срыта за прилегающими частями тела. Можно просто отключать видимость смежных объектов в области Outliner.
</callout>
## Тело
<callout icon="☑️" color="gray_bg">
	Оформите тело.
</callout>
- добавляем недостающие материалы и переименовываем их
- делим блок тела на нужное количество граней
- назначаем материалы соответствующим граням
<callout icon="💡" color="gray_bg">
	Часть материалов уже могла быть создана во время работы с блоком головы, например, кожа. Поэтому проверяйте созданные материалы и применяйте их не создавая дублей!
</callout>
## Руки
<callout icon="☑️" color="gray_bg">
	Оформите руки.
</callout>
- добавляем недостающие материалы и переименовываем их
- делим блок руки на нужное количество граней
- назначаем материалы соответствующим граням
<callout icon="💡" color="gray_bg">
	Обе руки могут иметь абсолютно одинаковое оформление. Когда вы оформите первую руку, ее можно просто продублировать на другую сторону, а старый экземпляр удалить.
</callout>
## Ноги
<callout icon="☑️" color="gray_bg">
	Оформите ноги.
</callout>
- добавляем недостающие материалы и переименовываем их
- делим блок ноги на нужное количество граней
- назначаем материалы соответствующим граням
<callout icon="💡" color="gray_bg">
	При использовании инструмента Loop Cut иногда удобно включать Snap Tool, чтобы примагничиваться к вершинам других объектов.
</callout>
