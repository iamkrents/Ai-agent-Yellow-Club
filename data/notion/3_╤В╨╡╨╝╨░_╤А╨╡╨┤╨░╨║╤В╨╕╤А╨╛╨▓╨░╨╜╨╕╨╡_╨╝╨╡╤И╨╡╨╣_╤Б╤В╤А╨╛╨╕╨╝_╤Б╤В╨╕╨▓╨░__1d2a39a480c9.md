# **Основы геометрии 3D-объектов**
<callout icon="☑️" color="gray_bg">
	Нарисуйте на доске куб и покажите из чего он состоит.
</callout>
Каждый меш состоит из:
- вершины (Vertex) — точки
- рёбра (Edge) — линии между точками
- грани (Face) — плоские поверхности
# **Режим редактирования (Edit Mode)**
<callout icon="☑️" color="gray_bg">
	Разберите, как происходит переключение между режимом редактирования и объектным режимом.
</callout>
- переключаем режимы - Edit Mode/Object Mode
	- Tab
	- меню в левом верхнем углу
<callout icon="☑️" color="gray_bg">
	Разберите, как происходит переключение между вершинами, рёбрами, гранями.
</callout>
- переключаемся между вершинами, рёбрами, гранями
	- горячие клавиши 1 / 2 / 3
	- иконки в верхней панели
<callout icon="☑️" color="gray_bg">
	Попросите учеников подвигать вершины, растянуть рёбра, повернуть грани.
</callout>
- выделяем 5 минут для свободной практики на кубике
<callout icon="☑️" color="gray_bg">
	Попрактикуйтесь на обезъянке.
</callout>
- добавляем объект Suzanne
- практикуемся в Edit Mode: 
	- выделение
	- перемещение
	- масштабирование
	- вращение
# **Подготовка изображения-референса Стива**
<callout icon="💡" color="gray_bg">
	После каждого этапа показывайте пример, а далее давайте возможность повторить → **ученик работает по шагам**
</callout>
<callout icon="☑️" color="gray_bg">
	Добавьте референс Стива.
</callout>
- скачиваем изображение Стива
- переходим в вид по оси X
- вставляем изображение (Image → Reference)
# **Построение головы**
<callout icon="☑️" color="gray_bg">
	Создайте голову Стива.
</callout>
- добавляем куб или работаем с текущим
- подгоняем размер референса под голову Стива
- дублируем референс и разворачивем перпендикулярно по оси Y
- подгоняем размещение референса под голову Стива
# **Построение тела**
<callout icon="☑️" color="gray_bg">
	Создайте тело Стива.
</callout>
- дублируем куб головы (Shift + D)
- опускаем его ниже головы по оси Z
## Инструмент Snap Tool
<callout icon="☑️" color="gray_bg">
	Разберитесь с инструментом Snap Tool.
</callout>
- создаем 2 кубика
- объясняем работу магнита (Snap Tool)
- рассматриваем типы привязки (Vertex / Edge / Face)
- примагничиваем кубики друг другу с разных сторон
## Доработка тела
<callout icon="☑️" color="gray_bg">
	Привяжите тело к голове и отрегулируйте его размер.
</callout>
- примагничиваем тело к низу головы
- вытягиваем тело вниз по оси Z в Edit Mode
- переключаемся в вид по оси Y
- уменьшаем толщину тема в Object Mode
# **Создание рук**
<callout icon="☑️" color="gray_bg">
	Создайте руки стива Стива.
</callout>
- дублируем тело
- примагничиваем дубликат к телу справа
- в Edit Mode уменьшаем ширину руки
- в Edit Mode увеличиваем длину руки
- дублируем руку и примагничиваем с другой стороны
# **Создание ног**
<callout icon="☑️" color="gray_bg">
	Создайте основу ног Стива.
</callout>
- дублируем тело
- примагничиваем дубликат к нижней части тела
- в Edit Mode увеличиваем длину ног
## Инструмент **Loop Cut**
<callout icon="☑️" color="gray_bg">
	Разберитесь с инструментом Loop Cut.
</callout>
- создаем куб
- переходим в Edit Mode
- объясняем и показываем, как работает инструмент Loop Cut (Ctrl + R)
- разрезаем на несколько частей куб
## Доработка ног
<callout icon="☑️" color="gray_bg">
	Сделайте 2 ноги Стиву.
</callout>
- разрезаем тело вертикально на 2 части посередине
- уменьшаем ширину блока ног в 2 раза, примагнитив к телу
- дублируем ногу и примагничиваем со второй стороны
