### Что такое массив?
Массив представляет набор однотипных данных. Формальное определение массива выглядит следующим образом `тип_переменной название_массива [длина_массива]`
Массив — это **набор элементов одного типа**, расположенных в памяти подряд.
```c++
int a[5];
```
Данный массив имеет пять чисел, но все эти числа имеют неопределенное значение. Чтобы установить значения элементов массива:
```c++
int a[5] = {1, 2, 3, 4, 5};
```
При этом не допускается присвоение одному массиву другого массива:
```c++
int nums1[] = {1,2,3,4,5};
int nums2[] = nums1;    // ошибка
```
**Рассказываем про индексы, выводим какие-то элементы массивов по индексам в консоль.**
Создаем массив из пяти элементов, но не заполняем его значениями `int a[5]`, пробуем вывести какой-то его элемент по индексу. Создаем еще один массив но не заполняем его до конца, например `int a[5] = {1, 2};` , а выведем например элемент под индексом 3 или 4. Пробуем выйти за границы массива
```c++
// Создаём массив из 5 чисел
    int a[5];                    // просто объявили
    int b[] = {10, 20, 30, 40, 50}; // сразу с значениями
    int b[5] = {};
    int c[] = {10, 20, 30}; // размер определяется автоматически
    int c[10] = {1, 2, 3};       // остальные будут нули

    // Индексы всегда начинаются с 0!
    cout << b[0] << endl; // 10
    cout << b[1] << endl; // 20
    cout << b[4] << endl; // 50
    // cout << b[5] << endl; // ОШИБКА! Выход за границы
```
Тут же показываем что мы можем перезаписывать элементы массива при обращении по индексу
```c++
int numbers[4] = {1,2,3,4};
int first = numbers[0];     // получаем первый элемент
std::cout << first << std::endl;    // 1
numbers[0] = 34;                    // изменяем значение элемента
std::cout << numbers[0] << std::endl; // 34
```
### Константные массивы
Если необходимо, чтобы нельзя было изменять значения элементов массива, то такой массив можно определить как **константный** с помощью ключевого слова **const**
```c++
const int numbers[4] = {1,2,3,4};
// numbers[1] = 23;    // ошибка - значения элементов массива изменить нельзя
```
Даем ребятам время самостоятельно подумать, как можно вывести весь массив целиком, а не только под одному элементу. Пускай попробуют, поэкспериментируют 
```c++
for(int i = 0; i < 5; i++) {
    cout << a[i] << " ";
}
```
Потом пускай самостоятельно подумают на вводом
```c++
for(int i = 0; i < 5; i++) {
    cout << a[i] << " ";
}
```
### Длина массива
Длина массива не всегда бывает известна. Однако может потребоваться получить ее. Для этого можно использовать несколько способов. Первый способ, который пришел из языка С, представляет применение оператора **`sizeof()`****.** Пускай ребята выведут на экран и попробуют понять что выводит этот метод
```c++
int a[3] = {1, 2, 3}
cout << sizeof(a); // длина массива в байтах 
```
Все элементы представляют один и тот же тип и занимают один и тот же размер в памяти. Поэтому мы можем с помощью выражения `sizeof(a)` найти длину всего массива в байтах, а с помощью выражения `sizeof(a[0])` - длину одного элемента в байтах. Разделив два значения, можно получить количество элементов в массиве. (издевательство - можетбыть)
<br>Второй способ представляет применение встроенной библиотечной функции **std::size()**:
```javascript
int numbers[] = {11, 12, 13, 14};
int count = std::size(numbers);
std::cout << "Length: " << count << std::endl;   // Length: 4
```

### Базовые алгоритмы с одномерным массивом
**Поиск максимума и минимума**
Даем простую задачу без замысловатого условия. Любые пять рандомных чисел на доске записываем и просим написать алгоритм для вывода максимального числа
```javascript
int maxValue = a[0];
for(int i = 1; i < 5; i++) {
    if(a[i] > maxValue)
        maxValue = a[i];
}
cout << "Max = " << maxValue;
```
**Поиск элемента (линейный поиск)**<br>Задача: Пользователь вводит 5 чисел и еще одно число. Вывести позицию найденного числа или фразу что число не найдено 
```javascript
int x;
cin >> x;
bool found = false;

for(int i = 0; i < 5; i++) {
    if(a[i] == x) {
        cout << "Нашелся на позиции " << i;
        found = true;
        break;
    }
}

if(!found) cout << "Не найден";
```
**Сортировка пузырьком!!! Пускай на доске пишут, на листике, пускай расписывают пока не поймут как работает!! Что б без подсказок могли бы потом реализовать**
```javascript
int n = 5;
for(int i = 0; i < n - 1; i++) {
    for(int j = 0; j < n - 1 - i; j++) {
        if(a[j] > a[j+1]) {
            int tmp = a[j];
            a[j] = a[j+1];
            a[j+1] = tmp;
        }
    }
}
```
Пояснить:
- два цикла
- сравниваем соседние
- большие элементы "всплывают" вправо

### Задачи на одномерные массивы
## <span color="green_bg">**1. Найти сумму элементов массива**</span>
**Условие:**
Дан массив из `n` чисел. Найдите сумму всех элементов.
### Решение
```c++
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    int a[100];
    for (int i = 0; i < n; i++) cin >> a[i];

    int sum = 0;
    for (int i = 0; i < n; i++) sum += a[i];

    cout << sum;
    return 0;
}
```
## <span color="orange_bg">**2. Найти максимальный элемент (можно попросить и макс и мин)**</span>
**Условие:**
Пользователь вводит массив из 10 чисел.
Выведите самое большое число в массиве.
### Решение
```c++
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    int a[100];
    for (int i = 0; i < n; i++) cin >> a[i];

    int mx = a[0];
    for (int i = 1; i < n; i++)
        if (a[i] > mx) mx = a[i];

    cout << mx;
    return 0;
}

```
## <span color="orange_bg">**3. Посчитать сумму чётных элементов**</span>
### Решение
```c++
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    int a[100];
    for (int i = 0; i < n; i++) cin >> a[i];

    int sum = 0;
    for (int i = 0; i < n; i++)
        if (a[i] % 2 == 0) sum += a[i];

    cout << sum;
    return 0;
}

```
## <span color="red_bg">**4. Развернуть массив (reverse)**</span>
### Решение
```c++
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    int a[100];
    for (int i = 0; i < n; i++) cin >> a[i];

    for (int i = 0; i < n / 2; i++) {
        int temp = a[i];
        a[i] = a[n - 1 - i];
        a[n - 1 - i] = temp;
    }

    for (int i = 0; i < n; i++) cout << a[i] << " ";
    return 0;
}
```
### **Двумерные массивы**
Объявление
```plain text
int A[n]
```
создает в памяти одномерный массив:  набор пронумерованных элементов, идущих в памяти последовательно. Но  можно создать и массив массивов следующим образом:
```plain text
int A[n][m]
```
Данное объявление создает массив из n объектов, каждый из которых в свою очередь является массивом типа
```plain text
int [m]
```
После типа переменной идет название массива, а затем в квадратных скобках его размер. 
Подобные объекты (массивы массивов) также называют двумерными массивами в С++. Двумерные массивы можно представлять в виде квадратной  таблицы, в которой первый индекс элемента означает номер строки, а  второй индекс — номер столбца. Например, массив
```plain text
A[3][4]
```
будет состоять из 12 элементов и его можно записать в виде

## Работа с двумерными массивами
Для считывания, вывода на экран и обработки двумерных массивов необходимо использовать вложенные циклы. Первый цикл — по первому индексу (то есть по всем строкам), второй цикл —  по второму индексу, то есть по всем элементам в строках. Например,  вывести на экран двумерный массив в виде таблицы, разделяя элементы в  строке одним пробелом можно следующим образом:
```javascript
int A[n][m];
for(int i = 0; i < n; ++i) {  // Выводим на экран строку i
    for(int j = 0; j < m; ++j)
        cout << A[i][j] << " ";
    cout << endl; // Строка завершается символом перехода на новую строку
}
```
А считать двумерный массив с клавиатуры можно при помощи еще более  простого алгоритма (массив вводится по строкам, то есть в порядке,  соответствующему первому примеру):
```javascript
for(int i = 0; i < n; ++i)
    for(int j = 0; j < m; ++j)
        cin >> A[i][j];
```
## Обработка одномерного массива
Обработка массивов производится аналогичным образом. Например, если  мы хотим записать в массив таблицу умножения, то есть присвоить элементу
```plain text
A[i][j]
```
значение i \* j, это можно сделать следующим образом:
```javascript
for(int i = 0; i < n; ++i)
    for(int j = 0; j < m; ++j)
        A[i][j] = i * j;
```
## Задачи
## **1. Вывести матрицу на экран**
**Условие:**
Дана матрица n×m. Выведите её в виде таблицы.
**Решение:**
```javascript
#include <iostream>
using namespace std;

int main() {
    int n, m;
    cin >> n >> m;

    int A[100][100];
    for (int i = 0; i < n; i++)
        for (int j = 0; j < m; j++)
            cin >> A[i][j];

    for (int i = 0; i < n; i++) {
        for (int j = 0; j < m; j++)
            cout << A[i][j] << " ";
        cout << endl;
    }
    return 0;
}

```
