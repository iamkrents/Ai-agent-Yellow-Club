# Способы подключения CSS
- повторить 3 метода
	- встроенный
	- внутренний
	- внешний
# Внутренний стиль (inline)
## Тег \<style\>
- разбираем тег и размещаем его в коде
## Свойства элементов
- разбираем куда и как записываются свойства
- переносим свойства 1 элемента
## Селекторы тега
- разбираем, как происходит привязка свойств к элементам
- привязываем свойства 1 элемента к тегу
- переносим все свойства в тег \<style\>
## Селекторы класса
- разбираем как добавляются селекторы класса
- разбираем атрибут class
- привязываем свойства текста через селектор класса
