## Функция вывода данных
Одной из задач программирования является обработка информации, поэтому языки программирования имеют средства для ввода и вывода информации.
Операторы ввода-вывода — это часть программы, которая инструктирует компьютер, как считывать и обрабатывать данные. Это относится к сбору информации с устройства ввода (клавиатуры, мыши, сканера, джойстика, микрофона  и т.п.) или отправке информации на устройство вывода (монитор, колонки, наушники, принтеры и т.п.). Синтаксис операторов ввода-вывода будет отличаться в зависимости от того, какой язык программирования используется.

## Библиотека ввода и вывода данных
Для работы с функцией вывода и ввода данных в языке С нужно подключить библиотеку.
<callout icon="🧑‍🎓" color="gray_bg">
	Зачем нужны библиотеки в языках программирования?
	<callout icon="🙋‍♂️" color="yellow_bg">
		Чтобы работать с готовыми функциями.
	</callout>
</callout>
В языке С ввод и вывод информации осуществляется через функции стандартной библиотеки:
<span color="blue">**\<stdio.h\>**</span> - библиотека для ввода/вывода данных
<span color="blue">**#include**</span> - конструкция для подключения библиотеки
```c
# include <stdio.h>
```

Для начала создаем структуру программы: подключаем библиотеку ввода и вывода данных и создаем точку входа в программу.
```c
// Напишите алгоритм для вывода в консоль текста с названием вашего любимого фильма/игры.

// подключение библиотеки ввода и вывода данных
# include <stdio.h> 

int main()                   // точка входа 
{                    
	
	return 0;
}
```

```c
// Напишите алгоритм для вывода в консоль текста с названием вашего любимого фильма/игры.
```
