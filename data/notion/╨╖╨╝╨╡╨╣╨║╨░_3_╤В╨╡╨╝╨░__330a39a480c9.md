Сегодня мы добавим:
- столкновение змейки **с самой собой**
- **поедание яблок**
- **рост змейки**
- **движение всего тела змейки**
Вспоминаем где находится код который считывает **следующую позицию головы**.<br>
```c++
int new_x =array_X[0]+dx;
int new_y =array_Y[0]+dy;
```
### Что здесь происходит
`array_X[0]` и `array_Y[0]` — это координаты **головы змейки**.
`dx` и `dy` — направление движения.
Например:
<table header-row="true">
<tr>
<td>Направление</td>
<td>dx</td>
<td>dy</td>
</tr>
<tr>
<td>вправо</td>
<td>1</td>
<td>0</td>
</tr>
<tr>
<td>влево</td>
<td>-1</td>
<td>0</td>
</tr>
<tr>
<td>вверх</td>
<td>0</td>
<td>-1</td>
</tr>
<tr>
<td>вниз</td>
<td>0</td>
<td>1</td>
</tr>
</table>
Поэтому новая позиция головы — это **текущая позиция + направление**.
Далее мы проверяем, **не врезалась ли змейка в стену**.
```c++
if (new_x<=0||new_x>=width-1||
new_y<=0||new_y>=height-1)
{
game_running =false;
continue;
}
```
### Почему именно такие числа
Поле ограничено рамкой `#`.
Например по X:
```plain text
0 #####################
1                     #
2                     #
...
49 ####################
```
Поэтому если змейка дошла до:
```plain text
0
или
width-1
```
значит она **врезалась в стену**.
`game_running = false` — останавливаем игру.
`continue` — сразу переходим к следующему циклу.
Пришло время добавить что-то новое в код СРАЗУ ПОСЛЕ ЭТОЙ ИФКИ
# Проверяем столкновение с собой
Теперь проверим **самую важную механику змейки**.
Змейка может **врезаться в собственное тело**.
```c++
// ─── ВРЕЗАЛИСЬ В СЕБЯ? ────────────────────────────────────────────────
boolhit_self =false;

for (int i =1;i<length;i++) // проверяем со 2-го звена и дальше
{
if (new_x==array_X[i]&&new_y==array_Y[i])
    {
			hit_self =true;
			break;
    }
}
```
### Как это работает
Мы проверяем:
```plain text
совпадает ли новая позиция головы
с координатами любого сегмента тела
```
Почему начинаем с `i = 1`?
Потому что:
```plain text
0 — голова
1 — первый сегмент тела
2 — второй сегмент
...
```
Если нашли совпадение:
```plain text
голова врезалась в тело
```
---
Теперь завершаем проверку:
```c++
if (hit_self)
{
game_running =false;
continue;
}
```
Если змейка врезалась в себя — игра заканчивается.
Но сейчас если запустить работать это не будет вот почему:
Сейчас в нашем коде проблема вот в чём:
Мы **сначала** проверяем, врезалась ли змейка в себя.<br>Если да → ставим game_running = false и пишем continue.
Но потом **всё равно** выполняем эти строки:
```c++
c.X = array_X[0]; cout << snake;          // затираем старую голову
array_X[0] = new_x;                       // двигаем голову
cout << head;                             // рисуем новую голову
```
То есть даже когда мы "сказали игре закончиться" — всё равно двигаем и рисуем змейку дальше.<br>Поэтому она продолжает ползти, хотя должна была остановиться.
Исправим это чуть позже, а для начала напишем следующее далее
# Проверяем съела ли змейка яблоко
Теперь проверяем **совпала ли голова с яблоком**.
```c++
// ─── СЪЕЛИ ЯБЛОКО? ────────────────────────────────────────────────────
boolate_apple = (new_x==X_apple&&new_y==Y_apple);
```
Это короткая запись:
```plain text
если координаты головы равны координатам яблока
→ ate_apple = true
```
# Если яблоко съели
```plain text
if (ate_apple)
{
length++;
}
```
Переменная `length` — это **длина змейки**.
Каждое яблоко делает её **на 1 сегмент длиннее**.Защита от переполнения массива
У нас есть массив:
```plain text
array_X[max_length]
array_Y[max_length]
```
Если змейка станет длиннее массива — программа сломается.
Поэтому добавляем защиту:
```c++
if (length>=max_length)
{
game_running =false;
continue;
}
```
В итоге сейчас у нас написано
```c++
        bool ate_apple = (new_x == X_apple && new_y == Y_apple);

        if (ate_apple)
        {
            length++;                           // змейка становится длиннее

            if (length >= max_length)           // защита от переполнения массива
            {
                game_running = false;
                continue;
            }
```
# Создаём новое яблоко
Теперь нужно поставить **новое яблоко**.
Но есть проблема: нельзя ставить яблоко **внутрь змейки**.
Поэтому используем цикл.
```plain text
bool position_ok;
do
{
X_apple =rand()% (width-2)+1;
Y_apple =rand()% (height-2)+1;
}
```
Это создаёт **случайные координаты внутри поля**.
# Проверяем, не попали ли на змейку
```plain text
position_ok =true;

for (inti =0;i<length;i++)
{
if (X_apple==array_X[i]&&Y_apple==array_Y[i])
    {
position_ok =false;
break;
    }
}
```
Если координаты совпали с телом змейки — значит место плохое.
Тогда цикл **повторится и найдёт новую позицию**.
# Цикл повторяется пока позиция плохая
```plain text
}while (!position_ok);
```
То есть:
```plain text
генерируем яблоко
проверяем
если совпало со змейкой → пробуем снова
```
В итоге этот кусок с яблоком выглядит так
```c++

        // ─── СЪЕЛИ ЯБЛОКО? ────────────────────────────────────────────────────
        bool ate_apple = (new_x == X_apple && new_y == Y_apple);

        if (ate_apple)
        {
            length++;                           // змейка становится длиннее

            if (length >= max_length)           // защита от переполнения массива
            {
                game_running = false;
                continue;
            }

            // ─── Ставим новое яблоко (не на змейке) ───────────────────────────
            bool position_ok;
            do
            {
                X_apple = rand() % (width - 2) + 1;
                Y_apple = rand() % (height - 2) + 1;

                position_ok = true;
                for (int i = 0; i < length; i++)
                {
                    if (X_apple == array_X[i] && Y_apple == array_Y[i])
                    {
                        position_ok = false;
                        break;
                    }
                }
            } while (!position_ok);
```

# Если яблоко не съели - стираем хвост
Если змейка **не съела яблоко**, она должна **двигаться**, а не расти.
Поэтому нужно **стереть последний сегмент хвоста**.
```plain text
c.X =array_X[length-1];
c.Y =array_Y[length-1];
SetConsoleCursorPosition(h,c);
cout<<' ';
```
Это создаёт эффект движения.
Сейчас у нас этот участок выглядит так
```c++
        // ─── СЪЕЛИ ЯБЛОКО? ────────────────────────────────────────────────────
        bool ate_apple = (new_x == X_apple && new_y == Y_apple);

        if (ate_apple)
        {
            length++;                           // змейка становится длиннее

            if (length >= max_length)           // защита от переполнения массива
            {
                game_running = false;
                continue;
            }

            // ─── Ставим новое яблоко (не на змейке) ───────────────────────────
            bool position_ok;
            do
            {
                X_apple = rand() % (width - 2) + 1;
                Y_apple = rand() % (height - 2) + 1;

                position_ok = true;
                for (int i = 0; i < length; i++)
                {
                    if (X_apple == array_X[i] && Y_apple == array_Y[i])
                    {
                        position_ok = false;
                        break;
                    }
                }
            } while (!position_ok);

            // сразу рисуем новое яблоко
            c.X = X_apple;
            c.Y = Y_apple;
            SetConsoleCursorPosition(h, c);
            SetConsoleTextAttribute(h, 12);
            cout << apple;
            SetConsoleTextAttribute(h, 10);     // вернули цвет змейки
        }
        else
        {
            // ─── СТИРАЕМ ХВОСТ (только если НЕ съели яблоко) ──────────────────
            c.X = array_X[length - 1];
            c.Y = array_Y[length - 1];
            SetConsoleCursorPosition(h, c);
            cout << ' ';
        }
```
# Сдвигаем всю змейку
Теперь нужно **сдвинуть тело змейки**.
Каждый сегмент занимает место предыдущего.
```c++
        // ─── СДВИГАЕМ ВСЮ ЗМЕЙКУ (хвост → голова) ────────────────────────────
        // Важно! Идём с конца, чтобы не затереть данные
for (inti =length-1;i>=1;i--)
{
array_X[i] =array_X[i-1];
array_Y[i] =array_Y[i-1];
}
```
Важно идти **с конца массива**, иначе мы перезапишем данные.
# Создаём новую голову
Теперь ставим новую позицию головы.
```plain text
array_X[0] =new_x;
array_Y[0] =new_y;
```
Этот кусок кода у нас уже написан, его просто нужно перенести выше
чтобы было
```c++
        // ─── СДВИГАЕМ ВСЮ ЗМЕЙКУ (хвост → голова) ────────────────────────────
// Важно! Идём с конца, чтобы не затереть данные
        for (int i = length - 1; i >= 1; i--)
        {
            array_X[i] = array_X[i - 1];
            array_Y[i] = array_Y[i - 1];
        }

        // обновляем координаты головы
        array_X[0] = new_x;
        array_Y[0] = new_y;


        // ─── Двигаем голову (пока без тела) ───────────────────────────────
        // затираем старую голову — делаем её обычным кусочком тела
        c.X = array_X[0];
        c.Y = array_Y[0];
        SetConsoleCursorPosition(h, c);
        cout << snake;                  // была голова — стала тело

```

```c++
        // ─── Двигаем голову (пока без тела) ───────────────────────────────
```

# Код целиком
```c++
#include <iostream>
#include <windows.h>
#include <conio.h>
using namespace std;

int main() {
    srand(time(0));
    system("title Snake Game");
    system("mode con cols=70 lines=31");
    MoveWindow(GetConsoleWindow(), 50, 50, 800, 600, true);

    const int width = 50, height = 30;
    const int max_length = 50;

    int array_X[max_length];
    int array_Y[max_length];
    int length = 1;

    array_X[0] = width / 2;
    array_Y[0] = height / 2;

    int dx = 1, dy = 0;         // ← начинаем двигаться вправо

    int X_apple, Y_apple;
    do {
        X_apple = rand() % (width - 2) + 1;
        Y_apple = rand() % (height - 2) + 1;
    } while (X_apple == array_X[0] && Y_apple == array_Y[0]);

    char snake = '*';
    char head = 2;
    char apple = 'o';

    COORD c;
    HANDLE h = GetStdHandle(STD_OUTPUT_HANDLE);

    CONSOLE_CURSOR_INFO cci = { sizeof(cci), false };
    SetConsoleCursorInfo(h, &cci);

    // ─────────────── Рамка и начальная картинка (это уже было на прошлом уроке) ───────────────
    SetConsoleTextAttribute(h, 4);
    for (int y = 0; y < height; y++) {
        for (int x = 0; x < width; x++) {
            if (x == 0 || x == width - 1 || y == 0 || y == height - 1)
                cout << '#';
            else
                cout << ' ';
        }
        cout << "\n";
    }

    c.X = X_apple; c.Y = Y_apple;
    SetConsoleCursorPosition(h, c);
    SetConsoleTextAttribute(h, 12);
    cout << apple;

    c.X = array_X[0]; c.Y = array_Y[0];
    SetConsoleCursorPosition(h, c);
    SetConsoleTextAttribute(h, 10);
    cout << head;

    // ───────────────────────────────────────────────────────────────
    // ↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓  НОВЫЙ КОД — НАЧИНАЕМ ПИСАТЬ ЗДЕСЬ  ↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓
    // ───────────────────────────────────────────────────────────────

    bool game_running = true;           // флаг "игра ещё идёт"
    int sleep_time = 120;               // сколько миллисекунд ждать между шагами (чем меньше — тем быстрее)

    // Самый главный цикл игры — будет крутиться, пока не проиграем
    while (game_running) {

        Sleep(sleep_time);              // ← делаем маленькую паузу, чтобы змейка не летала как ракета

        // ─── Проверяем, нажал ли игрок клавишу ───────────────────────────────
        if (_kbhit()) {                 // если в очереди есть хоть одна клавиша
            int k = _getch();           // берём первую часть кода

            // стрелки дают два байта — иногда сначала 0 или 224, потом настоящий код
            if (k == 0 || k == 224) {
                k = _getch();           // берём вторую часть
            }

            // теперь смотрим, какая именно клавиша
            switch (k) {
            case 72:    // стрелка вверх
                if (dy != 1) {      // запрещаем разворот на 180° (нельзя сразу вниз)
                    dy = -1;
                    dx = 0;
                }
                break;

            case 80:    // вниз
                if (dy != -1) {
                    dy = 1;
                    dx = 0;
                }
                break;

            case 75:    // влево
                if (dx != 1) {
                    dx = -1;
                    dy = 0;
                }
                break;

            case 77:    // вправо
                if (dx != -1) {
                    dx = 1;
                    dy = 0;
                }
                break;

            case 27:    // ESC — выход
                game_running = false;
                break;
            }
        }

        // ─── КУДА ПОЙДЁТ ГОЛОВА ───────────────────────────────────────────────
        int new_x = array_X[0] + dx;
        int new_y = array_Y[0] + dy;

        // ─── УДАРИЛИСЬ О СТЕНУ? ───────────────────────────────────────────────
        if (new_x <= 0 || new_x >= width - 1 ||
            new_y <= 0 || new_y >= height - 1)
        {
            game_running = false;
            continue;               // сразу переходим к следующей итерации цикла
        }

        // ─── ВРЕЗАЛИСЬ В СЕБЯ? ────────────────────────────────────────────────
        bool hit_self = false;
        for (int i = 1; i < length; i++)    // проверяем со 2-го звена и дальше
        {
            if (new_x == array_X[i] && new_y == array_Y[i])
            {
                hit_self = true;
                break;
            }
        }
        if (hit_self)
        {
            game_running = false;
            continue;
        }

        // ─── СЪЕЛИ ЯБЛОКО? ────────────────────────────────────────────────────
        bool ate_apple = (new_x == X_apple && new_y == Y_apple);

        if (ate_apple)
        {
            length++;                           // змейка становится длиннее

            if (length >= max_length)           // защита от переполнения массива
            {
                game_running = false;
                continue;
            }

            // ─── Ставим новое яблоко (не на змейке) ───────────────────────────
            bool position_ok;
            do
            {
                X_apple = rand() % (width - 2) + 1;
                Y_apple = rand() % (height - 2) + 1;

                position_ok = true;
                for (int i = 0; i < length; i++)
                {
                    if (X_apple == array_X[i] && Y_apple == array_Y[i])
                    {
                        position_ok = false;
                        break;
                    }
                }
            } while (!position_ok);

            // сразу рисуем новое яблоко
            c.X = X_apple;
            c.Y = Y_apple;
            SetConsoleCursorPosition(h, c);
            SetConsoleTextAttribute(h, 12);
            cout << apple;
            SetConsoleTextAttribute(h, 10);     // вернули цвет змейки
        }
        else
        {
            // ─── СТИРАЕМ ХВОСТ (только если НЕ съели яблоко) ──────────────────
            c.X = array_X[length - 1];
            c.Y = array_Y[length - 1];
            SetConsoleCursorPosition(h, c);
            cout << ' ';
        }

        // ─── СДВИГАЕМ ВСЮ ЗМЕЙКУ (хвост → голова) ────────────────────────────
        // Важно! Идём с конца, чтобы не затереть данные
        for (int i = length - 1; i >= 1; i--)
        {
            array_X[i] = array_X[i - 1];
            array_Y[i] = array_Y[i - 1];
        }

        // ─── Новая голова ─────────────────────────────────────────────────────
        array_X[0] = new_x;
        array_Y[0] = new_y;

        // ─── Перерисовываем ───────────────────────────────────────────────────
        // Предыдущая голова становится обычным телом
        if (length > 1)
        {
            c.X = array_X[1];
            c.Y = array_Y[1];
            SetConsoleCursorPosition(h, c);
            cout << snake;
        }

        // Рисуем новую голову
        c.X = array_X[0];
        c.Y = array_Y[0];
        SetConsoleCursorPosition(h, c);
        cout << head;
    }

    // ───────────────────────────────────────────────────────────────
    // ↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑  КОНЕЦ НОВОГО КОДА  ↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑
    // ───────────────────────────────────────────────────────────────

    system("cls");
    SetConsoleTextAttribute(h, 7);
    cout << "GAME OVER\n\n";

    return 0;
}
```
