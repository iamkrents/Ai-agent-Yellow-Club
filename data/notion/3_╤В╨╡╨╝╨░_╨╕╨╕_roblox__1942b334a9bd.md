# Регистрация/вход в Roblox studio
- Зайти на сайт [Roblox.com](http://Roblox.com)
- Создать новый аккаунт с годом рождения 18+ (запомнить логин и пароль)
- Зайти в приложение на компьютере Roblox studio и войти в аккаунт Roblox
# Обзор интерфейса Roblox Studio
- После открытия Roblox studio появились все шаблоны (All Templates)
Там находятся разные плейсы и примеры игр для роблокс и можно выбрать любой для ознакомления
- Первых 3 плейса - это пустые с разными платформами для разработки игры с нуля
- Мы выбираем 2 плейс для ознакомления интерфейса Roblox studio Classic baseplate
<span underline="true">**Знакомство с интерфейсом**</span>
- Home, Model, Avatar, Test, View, Plugins - тип редактора
- Верхняя панель - это свойства и инструменты редактора
- ToolBox - панель инструментов (готовые модели)
- Справа панель Explorer - обозреватель объектов
- По центру - рабочее пространство (Workspace)
- Properties - свойства объектов
<span underline="true">**Панель инструментов**</span>
- Select - выбор объектов
- Move - перемещение объекта
- Scale - изменение размера объекта
- Rotate - разворот объекта (Ctrl+R на 90гр.)
- Part - создание фигур (шар, цилиндр, рампа, блок)
- Material manager - материал для объектов
- Color - цвет объектов
- Anchor - якорь для объектов, чтобы закрепить
- Play - запуск предпросмотра
# Работа с объектами и создание дома (практика)
Чтобы закрепить навыки использования инструментов, мы потренируемся делать модель дома в роблокс
Для этого мы используем ИИ, чтобы сделать референс дома с помощью ruDall-e или Photosonic
Сгенерируем по запросу “roblox house simple small”
Выбрать подходящий референс и приступить к созданию дома
- Создадим первый блок Part
- Выстроим стены с помощью инструментов Move и Scale, Rotate
- Изменим материал наших стен, подходящим под референс Material Manager (кирпич, дерево, металл)
- Изменим цвет с помощью Color
- Добавить окна и дверь для дома можно с помощью ToolBox. В поиске ToolBox написать (лучше всего на английском языке) “window”, “door”
Обязательно! Нужно <span underline="true">закрепить</span> объекты дома с помощью <span underline="true">Anchor</span>, иначе, при запуске предпросмотра, дом развалится. 
Также, если нужно убрать <span underline="true">коллизию</span> к объекту, нужно через панель Properties найти <span underline="true">CanCollide</span> убрать галочку, тогда персонаж сможет проходить объект сквозь него.
