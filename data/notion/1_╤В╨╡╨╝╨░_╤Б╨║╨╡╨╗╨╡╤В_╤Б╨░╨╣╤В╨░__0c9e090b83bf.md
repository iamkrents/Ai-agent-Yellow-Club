При создании сайта мы общаемся не с человеком, а с системой/браузером. В языке HTML для общения мы будем использовать “теги” по аналогии с русскими “словами”.
# Синтаксис языка HTML
## Теги
Тег - это как слово в русском или другом языке, с помощью которого мы даём команды браузеру.
	- тег = слово
Для написания тегов используется латиница.
	- teg
Теги помещаются в треугольные скобки.
	- \<teg\>
## HTML-элементы
HTML-элемент - единица содержимого в HTML-документе, образованная HTML-тегами и содержащимся в нем текстом или медиафайлом. 
Синтаксис большинства HTML-элемента будет выглядеть следующим образом.
```html
<имя_тега>содержимое</имя_тега>
```
<span color="blue">**\<**</span><span color="blue">*имя_тега*</span><span color="blue">**\>**</span> - открывающий тег - используется для запуска HTML-элемента
<span color="blue">*cодержимое*</span> - информация (текст или другие элементы), содержащаяся между открывающим и закрывающим тегами HTML-элемента
<span color="blue">**\</**</span><span color="blue">имя_тега</span><span color="blue">**\>**</span> - закрывающий тег - используется для завершения HTML-элемента. Закрывающие теги имеют косую черту (/) внутри них, сразу после левой угловой скобки.
# Скелет сайта
## Тег <span color="blue">\<html\></span>
Первый тег:
```html
<html> 
```
Это контейнер, в который помещаются все элементы будущего сайта, то есть с помощью него мы даем команду начать сайта.
Разработчики для удобства сделали структуру сайта по образу человека.
<callout icon="👨‍🎓" color="gray_bg">
	Какие 2 основные части можно выделить у человека? 
	<callout icon="🙋‍♂️" color="yellow_bg">
		Тело и голову.
	</callout>
</callout>
У сайта тоже есть голова и тело. 
<callout icon="👨‍🎓" color="gray_bg">
	Как будет “голова” по-английски? 
	<callout icon="🙋‍♂️" color="yellow_bg">
		head
	</callout>
</callout>
## Тег <span color="blue">\<head\></span>
Следующий тег:
```html
<head> 
```
У головы человека есть место, где она начинается и где заканчивается. Так же и с головой сайта. С помощью тега <span color="blue">**\<head\>**</span> мы даем команду браузеру начать работу с головой. А как дать команду, что работа с головой была окончена? Мы просто дублируем тег и добавляем к нему символ “/”.
```html
</head> 
```
<callout icon="💡" color="gray_bg">
	Между открывающим и закрывающим тегом пропустим одну строку.
</callout>
```html
<html>
<head>

</head> 
```
<callout icon="👨‍🎓" color="gray_bg">
	Что находится в голове человека? 
	<callout icon="🙋‍♂️" color="yellow_bg">
		Мозги.
	</callout>
</callout>
И в голове сайта находятся его “мозги”. Тег <span color="blue">**\<head\>**</span> предназначен для хранения элементов, которые помогают браузеру в работе с данными. Поэтому почти всё, что мы укажем в теге <span color="blue">**\<head\>**</span> на сайте не будет отображаться.
## Тег <span color="blue">\<body\></span>
После головы идёт тело. 
<callout icon="👨‍🎓" color="gray_bg">
	Как по-английски будет “тело”? 
	<callout icon="🙋‍♂️" color="yellow_bg">
		body
	</callout>
</callout>
Следующий тег:
```html
 <body>
```
С помощью тега <span color="blue">**\<body\>**</span> мы даем команду браузеру начать работу с телом сайта.
<callout icon="👨‍🎓" color="gray_bg">
	А как дать команду закончить работу с телом? 
	<callout icon="🙋‍♂️" color="yellow_bg">
		<span color="blue">**\</body\>**</span>
	</callout>
</callout>
<callout icon="💡" color="gray_bg">
	Между открывающим и закрывающим тегом пропустим одну строку.
</callout>
Тег <span color="blue">**\<body\>**</span> содержит элементы, которые отображаются на сайте.
<callout icon="👨‍🎓" color="gray_bg">
	Какие элементы на сайте мы видим? 
	<callout icon="🙋‍♂️" color="yellow_bg">
		Текст, кнопки, картинки.
	</callout>
</callout>
Теги для текста, кнопок и картинок мы в дальнейшем и будем размещать в теле сайта.
На следующей строке дадим команду закончить работу с сайтом.
<callout icon="👨‍🎓" color="gray_bg">
	Какой тег даст команду закончить работу с сайтом или закроет контейнер? 
	<callout icon="🙋‍♂️" color="yellow_bg">
		<span color="blue">**\</html\>**</span>
	</callout>
</callout>
У нас получилась следующая структура, можно назвать ее “скелетом” сайта:
```html
<html>
<head>

</head>
<body>

</body>
</html>
```
После этого сохраним документ, нажав на синюю дискету на панели инструментов. 
Далее откроем файл в в браузере, чтобы посмотреть на результат.

	<callout icon="🙋‍♂️" color="yellow_bg">

“Скелет” сайта не будет отображаться на веб-странице. Скелет человека мы тоже не видим, но можем сделать рентген, чтобы увидеть кости.
Давайте сделаем “рентген” нашей веб-страницы и посмотрим код в браузере. 
> *Нажимаем правой кнопкой мыши и выбираем “Просмотреть код”. Здесь мы увидим код, который написали в блокноте.*
## Оформление кода
У каждого языка есть правила оформления кода. Эти правила нужны для более комфортного восприятия кода. Если какой-то тег находится между другими открывающим и закрывающим тегами, то внутренний тег мы выделяем отступом. Отступ можно сделать с помощью кнопки “Tab”.
<callout icon="👨‍🎓" color="gray_bg">
	Есть ли у нас теги, которые вложены друг в друга? 
	<callout icon="🙋‍♂️" color="yellow_bg">
		Да, \<head\>, \</head\>, \<body\> и \</body\> находятся между открывающим тегом \<html\> и закрывающим тегом \</html\>
	</callout>
</callout>
Давайте добавим к ним отступы.
```html
<html>													      
	<head>														

	</head>															
	<body>														

	</body>														  
</html>															  
```

Давайте напишем первый комментарий для тега <span color="blue">**\<html\>**</span>**.**

# Текст
## Тег <span color="blue">\<p\></span>
Теперь пройдём тег, который будет выводить текст на страницу.
Тег <span color="blue">**\<p\>**</span> дает команду создать абзац и поместить туда какой-либо текст. С помощью него мы можем добавить обычный текст на нашу веб страницу. 
<callout icon="👨‍🎓" color="gray_bg">
	Этот текст мы будем видеть на сайте. Значит куда мы добавим этот тег в \<head\> или в \<body\>?
	<callout icon="🙋‍♂️" color="yellow_bg">
		В \<body\>
	</callout>
	Например, я хочу, чтобы на сайте у меня отобразилось слово ”Привет”. Где мне его нужно написать в коде?
	<callout icon="🙋‍♂️" color="yellow_bg">
		Между \<p\> и \</p\>
	</callout>
</callout>
Верно, содержимое мы добавляем между открывающим и закрывающим тегом.
```html
<p>Привет</p>
```
Давайте пробовать.
<callout icon="👨‍🎓" color="gray_bg">

```html
	</body>															<!--Закончить работу с телом страницы-->

```
<callout icon="☑️" color="gray_bg">
