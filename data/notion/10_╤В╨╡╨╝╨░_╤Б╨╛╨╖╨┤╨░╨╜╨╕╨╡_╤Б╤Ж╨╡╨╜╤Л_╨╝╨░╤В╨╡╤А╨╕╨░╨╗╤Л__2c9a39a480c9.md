# Введение в Shader Editor
## Рабочее пространство Shading
<callout icon="☑️" color="gray_bg">
	Разберите рабочее пространство Shading.
</callout>
- переходим в рабочее пространство Shading
- разбираем открытые области
	- File Browser
	- 3D Viewport
	- Outliner
	- Properties
	- Image Editor
	- Shader Editor
## Shader Editor и ноды
<callout icon="☑️" color="gray_bg">
	Познакомьтесь с областью Shader Editor и разберите структуру стандартного материала в виде нод.
</callout>
- создаем и выбираем плоскость
- разбираем понятие ноды
- рассматриваем ноды материала плоскости
- меняем базовые настройки материала в ноде
	- Base Color
	- Roughness
	- Metallic
# Настройка текстур с помощью нод
## Поиск текстур
<callout icon="☑️" color="gray_bg">
	Скачайте и импортируйте объект с текстурами.
</callout>
- переходим на сайт Poly Haven
- ищем и скачиваем любой объект с текстурой
- открываем скачанный объект и переходим в Shading
## Разбор текстур материала
<callout icon="☑️" color="gray_bg">
	Разберите ноды текстур, из которых состоит материал.
</callout>
- открываем материал скачанного объекта и переходим в Shading
- разбираем, как работает и за что отвечает каждая нода группы Textures
	- Base Color - цвет
	- Roughness - шероховатость
	- Normal - объем
	- Displacement - искажение
## Трансформация текстур
<callout icon="☑️" color="gray_bg">
	Научитесь трансформировать текстуры.
</callout>
- разбираем, как работает и за что отвечает каждая нода группы Mapping
	- Mapping -

<callout icon="☑️" color="gray_bg">
	Создайте объект и добавьте ноды вручную.

- создаем плоскость
- переходим в Shader Editor

# Настройка материалов для объектов сцены
## Поиск текстур
<callout icon="☑️" color="gray_bg">
	Найдите и скачайте материалы для объектов сцены.
</callout>
- переходим на сайт Poly Haven
- ищем и скачиваем подходящие текстуры для объектов:
	- песок
	- лодка
	- цепь
	- пирс
	- камни
	- ствол пальмы
	- кокос
	- бочка
## Применение текстур к объектам
<callout icon="☑️" color="gray_bg">
	Примените и настройте материалы к объектам.
</callout>
- импортируем объект с текстурой объекта
- добавляем материал к объекту
- настраиваем размер и расположение текстуры
## Настройка материала воды
<callout icon="☑️" color="gray_bg">
	Создайте и настройте материал воды.
</callout>
- создаем новый материал
- настраиваем цвет
- устанавливаем Roughness на минимальное значение для создания отражения
- настраиваем параметр Alpha для добаления прозрачности воды
