# Условия задачи
<callout icon="✅" color="gray_bg">
	Изучить условия задачи и создать проект.
</callout>
Создать приложение, которое предоставит пользователю возможность сыграть в викторину.
**Перспектива**
Ребята смогут сделать из этого макета приложение для тренировки слов иностранного языка, столиц стран, подготовки к каким-то работам. Они могут добавить картинки, рандомную генерацию вопросов, повторение вопросов, на которые не ответили правильно с первого раза.
**Дизайн**
- на главном экране расположена шапка с надписью “Викторина”
- справа от надписи размещается надпись “Баллы: 0”
- под шапкой находится вопрос
- под вопросом располагается поле для ввода ответа на вопрос
- под полем кнопка с надписью “Ответить”
	- цвет кнопки - желтый
	- цвет текста - черный
**Функционал**
При запуске приложения появляется первый вопрос. После ввода ответа есть 2 варианта:
- если игрок ответил верно, то балл добавляется и появляется следующий вопрос
- если же игрок ответил неправильно, то всплывает уведомление о том, что ответ неверный и стоит попробовать снова и не дает игроку новый вопрос до тех пор, пока игрок не введет верный ответ. 
При ответе на все вопросы, кнопка гаснет и появляется уведомление об успешном ответе на все вопросы, а поле для ввода очищается.

# Логика приложения
<callout icon="✅" color="gray_bg">
	Прописать логику работы приложения.
</callout>
- объявляем переменные
- создаем обработчик событий 
```kotlin
package com.yxl.calculator

import android.os.Bundle
import android.widget.Button
import android.widget.EditText
import android.widget.TextView
import android.widget.Toast
import androidx.activity.enableEdgeToEdge
import androidx.appcompat.app.AppCompatActivity
import androidx.core.view.ViewCompat
import androidx.core.view.WindowInsetsCompat

class MainActivity : AppCompatActivity() {

    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        enableEdgeToEdge()
        setContentView(R.layout.activity_main)

        // Список вопросов
        val questions = listOf(
            "Столица Франции?",
            "2 + 2 = ?",
            "Как называется наша планета?"
        )
        // Список ответов на вопросы в таком же порядке
        val answers = listOf(
            "Paris",
            "4",
            "E"
        )
        // Создаем переменные для элементов верстки
        val tvQuestion: TextView = findViewById(R.id.tvQuestion)
        val etAnswer: EditText = findViewById(R.id.etAnswer)
        val bSubmit: Button = findViewById(R.id.bSubmit)
        val tvAnswer: TextView = findViewById(R.id.tvScore)
        var correctAnswers = 0
        // Игнорируем
        ViewCompat.setOnApplyWindowInsetsListener(findViewById(R.id.main)) { v, insets ->
            val systemBars = insets.getInsets(WindowInsetsCompat.Type.systemBars())
            v.setPadding(systemBars.left, systemBars.top, systemBars.right, systemBars.bottom)
            insets
        }

        var indexOfQuestion = 0 // Индекс текущего вопроса
        //Устанавливаем первый вопрос
        tvQuestion.text = questions[indexOfQuestion]

        // Устанавливаем обработчик событий на кнопку ответить
        bSubmit.setOnClickListener {
            val userAnswer = etAnswer.text.toString().trim() // Ответ пользователя
            // Верный ответ - элемент из списка ответов по индексу вопроса
            // нулевой вопрос - нулевой ответ и тд
            val correctAnswer = answers[indexOfQuestion]

            if (userAnswer == correctAnswer) {
                // Если ответ правильный, то мы увеличиваем индекс, а значит перейдем к следующему вопросу
                indexOfQuestion = indexOfQuestion + 1

                // Делаем счетчик правильных ответов больше на 1
                correctAnswers = correctAnswers + 1
                tvAnswer.text = "Баллы: $correctAnswers"

                if (indexOfQuestion < questions.size) {
                    // Переходим к следующему вопросу
                    tvQuestion.text = questions[indexOfQuestion]
                    etAnswer.text.clear()

                } else {
                    // Если вопросы закончились
                    Toast.makeText(this, "Вы прошли викторину!", Toast.LENGTH_LONG).show()
                    etAnswer.text.clear() // очищаем поле ввода ответа
                    bSubmit.isEnabled = false // Отключаем кнопку, не показывали ребятам, но ничего страшного в этом нет
                }
            } else {
                // Если ответ неверный
                Toast.makeText(this, "Неверный ответ! Попробуйте ещё раз.", Toast.LENGTH_SHORT).show()
            }
        }
    }
}


```
