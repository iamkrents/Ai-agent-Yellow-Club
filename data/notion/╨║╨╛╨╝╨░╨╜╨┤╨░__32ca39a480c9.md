# 👨‍👨‍👧‍👧 Организационная структура
Схема ниже показывает, как устроена команда и зоны ответственности внутри компании.
```kotlin
CEO / Основатель
│
├ Операционный менеджер
│   └ Преподаватели
│       (операционные вопросы: расписание, замены, классы, системы)
│
├ Ведущий клиент-менеджер
│   └ Клиент-менеджер
│
├ Старший преподаватель / методист
│   └ Преподаватели
│       (методика обучения, качество занятий, развитие преподавателей)
│
├ Маркетинг
│   ├ Маркетолог
│   └ SMM
│
└ Бухгалтер
```
# 💬 К кому обращаться
Ориентируйся на этот блок, чтобы быстро понять, кто отвечает за решение твоего вопроса.
⚙️ Операционный менеджер
-  нужно заменить преподавателя
-  вопрос по расписанию
-  проблема в учебном классе или оборудовании
-  не работает CRM / система / платформа
-  непонятно, как организован процесс
💬 Клиент-менеджеры
- родитель задаёт вопрос или недоволен
- ученик пропускает занятия
- нужно связаться с клиентом
- вопрос по заявкам
🎓 Старший преподаватель / методист
- не понимаешь, как провести занятие
- есть вопросы по программе или уроку
- хочешь улучшить занятие
🦄 Маркетолог / SMM
- есть идеи по рекламе или контенту
📂 Бухгалтер
- вопросы по оплатам, зарплате, документам
---
# 🤝 Как устроено взаимодействие
- сначала обращайся к своему прямому руководителю
- если вопрос не решился → поднимай выше по структуре
- в срочных ситуациях → пиши напрямую ответственному
Важно:
- не оставляй проблему без реакции
- чем раньше сообщил – тем проще решить
