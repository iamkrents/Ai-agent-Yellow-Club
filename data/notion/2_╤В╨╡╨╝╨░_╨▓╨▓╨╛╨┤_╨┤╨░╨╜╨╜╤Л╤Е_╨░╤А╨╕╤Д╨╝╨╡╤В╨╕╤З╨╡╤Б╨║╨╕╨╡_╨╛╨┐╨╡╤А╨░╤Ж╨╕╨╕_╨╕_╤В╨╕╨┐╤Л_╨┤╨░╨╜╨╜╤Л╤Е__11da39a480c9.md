// Вывод арифметических операций в консоль
println(3 + 5)

// Типы данных в Kotlin
// Integer (Int) - целые числа, например 0, 5, -5
// Float (Float) - дробные числа, например 2.5, 3.7, 2.0
// Double (Double) - дробные числа, могут хранить больше знаков после запятой
// Char (Char) - 1 символ, помещается в одинарные кавычки, например '5'
// String (String) - текст, помещается в двойные кавычки, например "привет"

// Объявление переменной, ее типа данных и присвоение значения
val a: Int 
a = 10

// Принято записывать объявление переменной/константы и присвоение значения в одну строку, если известно значение
val a: Int = 10
val b: Float = 10.4f      // При объявлении переменной типа Float в конце ставим букву f, если не поставим, будет Double
val c: Double = 12.3456
val d: Char = 'a'
val e: String = "Hello, world!"

// Какие арифметические операции можно делать с разными типами данных
// Int, Float, Double - все
// String и Char - только сложение со String

// String и Char можем только складывать со String - конкатенация

// Объявите отдельную переменную для записи результата
// Выведе результат 4 арифметических операций между 2 числами в консоль

res = a + b
println("a + b = $res")
res = a - b
println("a - b = $res")
res = a * b
println("a * b = $res")

// При выводе результата деления с остатком, остаток теряется, если изначально конвертация происходила в Int
res = a / b
println("a / b = $res")


// Скорректированный вариант, чтобы не терять дробную часть в результате деления
