# Регистрация аккаунта и настройка среды
## **Создание аккаунта**
1.	Откройте браузер и перейдите на сайт [Roblox.com](http://roblox.com/).
2.	На главной странице заполните поля:
- **Username (Имя пользователя)** - выберите уникальный ник
- **Password (Пароль)** - придумайте сложный пароль и запомните его
- **Date of birth (Дата рождения)** - выберите **год рождения 18+**, чтобы получить полный доступ к функциям
- **Gender (Пол)** - можно не указывать
3.	Нажмите кнопку **Sign Up (Зарегистрироваться)**.
4.	Пройдите проверку на робота (если появится).
5.	После регистрации **сохраните логин и пароль** в надежном месте.
## **Установка Roblox Studio**
1.	Войдите в аккаунт на сайте [Roblox.com](http://roblox.com/).
2.	В верхнем меню нажмите **Create** → **Start Creating**.
3.	Появится окно с предложением скачать Roblox Studio - нажмите **Download Studio**.
4.	Откройте скачанный файл и следуйте инструкциям по установке.
5.	После установки **запустите Roblox Studio** и войдите в свой аккаунт.
## Первый запуск
После входа в Roblox Studio откроется окно с шаблонами (All Templates).
Первые три шаблона - это пустые сцены для создания игры с нуля.
Выберите **Classic Baseplate**, чтобы изучить интерфейс.
# Интерфейс Roblox Studio
- **Workspace **- рабочее пространство (центральная область), где создается игра
- **Верхняя панель** с вкладками:
	- **Home** - базовые инструменты
	- **Model** - редактирование объектов
	- **Avatar** - работа с персонажами
	- **Test** - тестирование игры
	- **View** - включение и отключение окон
	- **Plugins** - установленные плагины
- **ToolBox** - панель инструментов (слева), где находятся готовые модели и объекты
- **Explorer** - обозреватель объектов (справа), в котором отображаются все элементы сцены
- **Properties** - свойства объектов (справа), где можно менять параметры выбранных объектов
# Работа с объектами
- **Play** - запуск предпросмотра
## **Добавление объектов**
Откройте вкладку **Home**.
Нажмите кнопку **Part** - появится объект (по умолчанию это блок).
Чтобы изменить форму объекта, нажмите **маленькую стрелку** рядом с кнопкой Part и выберите:
- **Block** - куб
- **Sphere** - сфера
- **Cylinder** - цилиндр
- **Wedge** - рампа
## Трансформация объектов
- **Select** - выбор объектов
Выберите объект с помощью инструмента **Select**.
Для редактирования используйте:
- **Move** - перемещение объекта (синяя, красная и зеленая стрелки)
- **Rotate** - вращение (красные, зеленые и синие кольца)
- **Scale** - изменение размера (цветные точки по углам)
## Настройка внешнего вида
Выберите объект.
В панели **Properties** можно изменить:
- **Color** - изменить цвет
- **Material** - выбрать материал (Grass, Metal, Wood и др.).
Включите **Material Manager**, чтобы посмотреть список доступных материалов.
## Дополнительные свойства
- **Anchor** - фиксирует объект на месте (без этого он может упасть)
- **Transparency** - делает объект прозрачным (от 0 до 1)
- **Collision** - управляет столкновениями (если отключить, объект можно проходить насквозь)
# Практика
## **Простая игровая площадка**
Сцена будет представлять собой небольшую платформу с базовыми объектами, на которой игрок сможет передвигаться.
**Шаги для создания:**
1.	**Основа сцены**
- добавить **блок (Part)** и увеличить его в размере (Scale), чтобы создать пол
- закрепить его с помощью **Anchor**, чтобы не проваливался.
- изменить цвет и материал через **Properties** (например, сделать зеленую траву или деревянную платформу).
2.	**Добавление препятствий и объектов**
- добавить несколько **блоков разного размера** в случайных местах (это могут быть колонны, трамплины, стены)
- разместить **рампу**, чтобы можно было забраться наверх
- добавить **сферу или цилиндр** и изменить его цвет (например, сделать «мяч» или декоративный элемент)
3.	**Тестирование сцены**
- запустить **Play** и проверить, как выглядят объекты
- использовать **Rotate и Move**, чтобы подправить расположение объектов.
**Дополнительно:**
- можно сделать **невидимую стену**, увеличив прозрачность объекта
- создать **прыгающую платформу**, изменив коллизию и тестируя взаимодействие
В итоге получится **небольшая тренировочная площадка**, где игрок сможет бегать, прыгать и изучать взаимодействие с объектами. Это даст базовые навыки для создания более сложных уровней.
## **Мини-замок или башня с лестницей**
Сцена будет представлять собой небольшую крепость или башню с лестницей, куда можно подняться. 
**Шаги для создания:**
1.	**Основа сцены (фундамент и стены)**
- создать **большой Part** и увеличить его (Scale) – это будет пол
- обавить **четыре высокие стены (Part)**, расставив их по краям, чтобы получилась комната или площадка
- закрепить стены **Anchor**, чтобы они не падали
- изменить **материал** на «Камень» или «Бетон», чтобы создать эффект замка.
2.	**Добавление лестницы**
- создать **несколько небольших блоков (Part)** и расположить их друг на друге в форме ступенек
- настроить **Scale** и **Move**, чтобы лестница вела на крышу или второй этаж
- закрепить **Anchor**, чтобы лестница не падала
3.	**Создание башни / второго этажа**
- добавить **площадку (Part)** наверху лестницы
- разместить **перила или стены** вокруг площадки
- можно добавить **цилиндры** в углах для украшения (например, башенки)
4.	**Декорации и тестирование**
- изменить цвета и материалы (например, стены серые, пол деревянный).
- добавить **флаг (Part + Color), факел или другие декоративные элементы**.
- проверить сцену через **Play**, убедиться, что по лестнице можно забраться
В итоге получится простая крепость или башня, по которой можно перемещаться. Это станет основой для будущих уровней или сложных построек.
