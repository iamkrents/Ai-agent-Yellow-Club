# Вводная часть
- рассказываем, какую игру мы будем делать (идея и концепция)
- объясняем, что такое хоррор-игра и чем она отличается от других жанров
- рассказываем, как будет выглядеть карта и какие зоны на ней будут
- поясняем, какие знания и навыки нам понадобятся
- фиксируем допустимые правила игры
# Ландшафт (2 урока)
- знакомимся с **Terrain Editor **и его инструментами
- создаем тестовую локации с помощью инструментов **Terrain Editor**
# Создание своей карты (2 урока)
## Блокинг
- делаем блокинг основных элементов локации:
	- горы
	- пещеры
	- туннели
	- озеро/ река
	- лес
	- объект точки финиша с дверью
	- стартовая точка
	- укрытия - чекпоинты
	- спаунеры зомби
	- скримеры
	- размещение цели - ключа
- проверяем проходимость карты
## Ландшафт
- реализуем ландшафт карты исходя из блокинга
# Моделирование объектов
- учимся работать с референсами
- делаем модель двери
- учимся работать с декалями
- учимся работать с текстурами
- рассматриваем отличие работы с папками и моделями
- делаем и оформляем модель ключа
# Создание инструментов
- учимся создавать инструменты 
	- из одного примитива
	- из нескольких примитивов
	- балуемся с инструментами
- создаем инструмент из ключа
- делаем иконку инструмента
# Работа со скриптами (1-2 урока)
- добавляем и разбираем скрипт “Hello World”
- создаем для смена цвета кубика
- создаем скрипт для смены дня и ночи
- разбираем и добавляем скрипты
	- дверь + ключ
- проверяем работу на карте
# GUI
- разбираем что такое GUI 
	- StarterGUI
	- ScreenGUI
	- TextLabel
- делаем часы
	- добавляем часы на экран
	- добавляем скрипт, чтобы часы работали
- делаем подсказку при подходе к двери
# Звук
- учимся работать со звуком
	- добавляем фоновую музыку
	- разбираемся с настройками звука
- делаем скримеры (условие + картинка + звук)
# NPC
- что такое NPC и как они работают в Roblox
- рассмотреть 2 способа создания NPC
	- из part
	- через инструмент character
- разбираем из чего состоит стандартный NPC
- создаем модель NPC из part
	- создаем модель из part
	- добавляем объект humanoid
	- корректно подписываем part
	- добавляем body color
	- добавляем одежду и детали
# Анимация персонажа
- разбираем инструмент Character
- разбираем разные модели и выбираю 1 для работы
- выбираем и добавляем одежду из toolbox и roblox
- учимся работать с clipboard для создания анимаций
	- разбираем интерфейс clipboard
	- создаем анимацию стойки для игрока
	- как правильно подключить анимацию
	- запускаем и проверяем анимацию
# Анимация NPC
- анимируем NPC (танец или доделываем анимацию с предыдущего занятия)
- реализуем анимацию ходьбы NPC - зомби
- анимация NPC для диалогов
# Диалоги
- разбираемся с dialog и dialogchoice
- тренируемся создавать диалоги в игровом пространстве
- продумываем, где и какие нужны диалоги по сюжету
- создаем диалоги персонажа с NPC
- проверяем работу диалогов
- создаем анимацию покоя NPC
- подключаем 2 анимации к NPC
- проверяем финальный результат
# Анимация зомби
- разбираем плагин EditRig+
- добавляем Motor6D
- добавляем анимацию на зомби
- добавляем скрипт
	- чтобы зомби следовал за персонажем
	- чтобы зомби наносил урон
	- чтобы ограничить зону
- проверяем проходимость карты
# 🟡Атфмосфера и доработка карты
## Свет
- меняем свет
- меняем небо
- добавляем туман
- добавляем другие эффекты
## Детали
- добавляем детали на карту
- доделываем
	- карту
	- механики
	- логику
# Тестирование и релиз
- проверяем карту на проходимость
- исправляем баги и дорабатываем
- делаем релиз игры
