# Арифметические операции
<callout icon="🧑‍🎓" color="gray_bg">
	Для чего нужны арифметические операции?
	<callout icon="🙋‍♂️" color="yellow_bg">
		Чтобы делать вычисления.
	</callout>
</callout>
Арифметические операции позволяют производить вычисления. Их можно применять практически ко всем встроенным типам данных.
## Арифметические операторы
Для выполнения арифметических операций используются операторы:
- + (сложение)
- - (вычитание)
- \* (умножение)
- / (деление)
- % (остаток от деления)
Они работают в языке  С точно так же, как и в большинстве других языков.
## Виды арифметических операций
Для примера объявим переменные <span color="blue">**a**</span> и<span color="blue"> </span><span color="blue">**b**</span>, которым присвоим целочисленные значения. А также переменную <span color="blue">**c**</span>, которая будет принимать результаты вычислений.
```c
#include <stdio.h>

int main()
{
	int a;
	a = 8;
	int b;
	b = 4;
	int c;

	return 0;
}
```
### **Сложение**
Операция сложения возвращает сумму двух чисел.
```c
#include <stdio.h>

int main()
{
	int a;
	a = 8;
	int b;
	b = 4;
	int c;
	
	c = a + b;
	printf("a + b = %d\n", c);

	return 0;
}
```
Результатом будет сложение 8 и 4, то есть 12.
### **Вычитание**
Операция вычитания возвращает разность двух чисел.
```c
#include <stdio.h>

int main()
{
	int a;
	a = 8;
	int b;
	b = 4;
	int c;
	
	c = a - b;
	printf("a - b = %d\n", c);

	return 0;
}
```
Результатом будет вычитание 8 и 4, то есть 4.
### **Умножение**
Операция умножения возвращает произведение двух чисел.
```c
#include <stdio.h>

int main()
{
	int a;
	a = 8;
	int b;
	b = 4;
	int c;
	
	c = a * b;
	printf("a * b = %d\n", c);

	return 0;
}
```
Результатом будет произведение 8 и 4, то есть 32.
### **Деление**
Операция деления двух чисел.
```c
#include <stdio.h>

int main()
{
	int a;
	a = 8;
	int b;
	b = 4;
	int c;
	
	c = a / b;
	printf("a / b = %d\n", c);

	return 0;
}
```
Результатом будет деление 8 и 4, то есть 2.
Когда  деление применяется  к целому числу, остаток обрубается. Например, при деление 16 на 5 результатом будет 3. Если в операции деления участвует хотя бы одно вещественное число, то результат будет вещественным.
### **Остаток от деления**
Возвращает результат деления двух чисел по модулю (получение остатка от деления).
```c
#include <stdio.h>

int main()
{
	int a;
	a = 16;
	int b;
	b = 5;
	int c;
	
	c = a % b;
	printf("Остаток от деления a на b равен %d\n", c);

	return 0;
}
```
Здесь без остатка на 5 будет делиться 15.  Это значит, что остатком от деления будет 1 (16 - 15 = 1). Поэтому результатом будет вывод в консоль цифры 1.
