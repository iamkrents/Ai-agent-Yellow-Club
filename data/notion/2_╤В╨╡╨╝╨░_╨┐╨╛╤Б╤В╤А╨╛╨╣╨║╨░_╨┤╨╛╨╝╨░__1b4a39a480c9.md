# **Создание основы дома**
- базовый уровень - построить простой дом с крышей, дверью и окнами
- средний уровень - добавить второй этаж, лестницу и балкон
- продвинутый уровень - cделать интерактивную дверь, которая открывается по нажатию
## Основание
- перейдите в верхнее меню **Home** → нажмите **Part** → появится стандартный куб
- выберите инструмент **Scale** (изменение размеров) и растяните **Part** вширь, чтобы создать основание дома
- откройте вкладку **Properties** (если её нет, включите через **View → Properties**)
- в разделе **Material** выберите подходящий материал:
	- **Grass** (трава) - если дом будет стоять на земле
	- **Concrete** (бетон) - если нужен фундамент
- при необходимости поменяйте цвет через **Properties → Color**
## **Постройка стен**
- нажмите **Home → Part**, чтобы добавить новый блок
- переместите его с помощью **Move** (инструмент для перемещения)
- вытяните вверх с помощью **Scale**, чтобы задать высоту стены
- откройте **Properties** и выберите **Material**:
	- **Brick** (кирпич) для классического дома
	- **Wood** (дерево) для деревянного дома
	- **Concrete** (бетон) для современного здания
- измените цвет стен через **Properties → Color**
# **Добавление дверей и окон**
## **Создание проёмов**
- добавьте новый **Part** там, где будет дверь или окно
- настройте его размер с помощью **Scale** (он должен соответствовать размеру проёма)
- выберите проём и нажмите **Model → Negate** (отрицательная форма)
- выделите стену и проём одновременно, затем нажмите **Model → Union** - появится вырез
## **Добавление и настройка окон**
- добавьте новый **Part** в проём окна
- измените его размеры с помощью **Scale**, чтобы он точно вписался в проём
- В **Properties**:
	- становите **Transparency → 0.5** (прозрачность)
	- выберите **Material → Glass** (стекло)
	- измените цвет стекла, например, на **голубой**
- если хотите добавить оконную раму:
	- создайте тонкие **Part**, обрамляющие стекло
	- выберите для них **Material → Metal** или **Wood**
## **Добавление дверей**
- создайте дверь с помощью **Part** (обычно немного тоньше стены)
- переместите её в дверной проём
- в **Properties**:
	- измените **Color** (например, коричневый для деревянной двери)
	- выберите **Material → Wood** или **Metal**
## Настройка дверей
### **Добавление петель**
Чтобы дверь открывалась, её нужно зафиксировать на петле.
- выберите дверь (Part, который будет открываться)
- переместите её так, чтобы одна сторона прилегала к краю дверного проёма
- убедитесь, что дверь не закреплена (в **Properties** параметр **Anchored** должен быть отключён)
- вставьте в дверь **Attachment**:
	- выделите дверь
	- в меню **Model → Create → Attachment** добавьте точку привязки
	- переместите **Attachment** на край двери (ту сторону, где будет петля)
- вставьте в стену рядом с дверью ещё один **Attachment**:
	- выделите стену рядом с дверью
	- создайте ещё один **Attachment** и разместите его на одном уровне с первым (вдоль оси вращения двери).
- добавьте **HingeConstraint** (петли):
	- перейдите в **Model → Constraints → HingeConstraint**
- в **Properties** укажите:
	- **Attachment0** → прикрепите к двери
	- **Attachment1** → прикрепите к стене
Теперь дверь закреплена на петле и может свободно вращаться.
### **Установка лимитов**
Чтобы дверь не проворачивалась бесконечно, добавим ограничения.
- выделите **HingeConstraint** в **Explorer**
- в **Properties** найдите **LimitsEnabled** и включите его (поставьте галочку)
- в полях **LowerAngle** и **UpperAngle** установите допустимые углы:
	- **LowerAngle = 0** (дверь закрыта)
	- **UpperAngle = 90** (дверь полностью открыта)
Теперь дверь не сможет открываться дальше 90 градусов.
### **Добавление пружины для возврата двери в исходное положение**
Чтобы дверь автоматически закрывалась после открытия, используем **SpringConstraint.**
- перейдите в **Model → Constraints → SpringConstraint**
- в **Explorer** перенесите **SpringConstraint** внутрь двери
- в **Properties** настройте:
	- **Attachment0** → прикрепите к двери
	- **Attachment1** → прикрепите к стене (там же, где HingeConstraint)
	- **FreeLength = 0** (дверь будет стремиться вернуться в начальное положение)
	- **Stiffness = 50** (жесткость пружины, можно настроить)
	- **Damping = 10** (демпфирование, чтобы дверь не дрожала)
Теперь дверь плавно открывается, когда персонаж толкает её, а затем автоматически закрывается.
# **Создание крыши**
- вставьте новую **Part** и переместите её наверх
- измените её форму с помощью **Scale** (растяните по размеру дома)
- для скатной крыши:
	- добавьте два **Wedge** (наклонные элементы)
	- переместите их и используйте **Rotate** (вращение), чтобы задать угол наклона
- выберите материал крыши:
	- **Wood** - для деревянной крыши
	- **Metal** - для металлической
	- **Tiles** - черепичная крыша
# **Улучшение внешнего вида**
- выделите элементы дома и в **Properties** измените их **Color**
- в **Material** задайте более реалистичные текстуры (кирпич, камень, бетон)
- добавьте мелкие детали:
	- лестницу перед входом (несколько **Part** разной высоты)
	- забор вокруг дома
	- дымоход (небольшой **Part**, размещённый на крыше)
