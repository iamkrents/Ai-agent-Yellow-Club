# Кнопка

С текстом мы с вами уже поработали и узнали, с помощью каких тегов можем его добавить.
<callout icon="👨‍🎓" color="gray_bg">

	Давайте теперь добавим кнопку. Как кнопка по-английски? 
	<callout icon="🙋‍♂️" color="yellow_bg">

	</callout>
</callout>

Давайте добавим кнопку после текста, напишем тег <span color="blue">**\<button\>**</span>.

Теперь добавим тег, который помогает добавить картинку. Это тег <span color="blue">**\<img/\>**</span>. 
<callout icon="👨‍🎓" color="gray_bg">
	Это непарный тег, значит как мы его напишем? 
	<callout icon="🙋‍♂️" color="yellow_bg">

	</callout>

Некоторые теги не могут работать самостоятельно, им нужны помощники. Эти помощники называются атрибутами. 
# Атрибуты
Атрибуты мы пишем внутри открывающегося тега. После артибута обязательно ставим знак равно и открываем кавычки.
```html
<tag атрибут=”источник”>
```
## Атрибут src
Мы хотим добавить картинку, значит тег <span color="blue">**\<img/\>**</span> будет выглядеть так:
```html
<img src=”источник”/>
```
Мы можем добавить картинку из интернета, но не стоит забывать, что у картинок есть авторские права, которые нельзя нарушать, поэтому лучше картинки искать на фотостоках.
Фотосток - лучше качество, не нарушаем авторские права.
Мы можем использовать такие саты как Pixabay и Unsplash. Выбираем изображение, нажимает правой кнопкой мыши и копируем URL картинки.

</callout>
Добавляем скопированную ссылку в тег <span color="blue">**\<img/\>**</span>.

# Храниение файлов
Ищем картинку и сохраняем на рабочий стол.
<callout icon="👨‍🎓" color="gray_bg">
	Как теперь эту картинку добавить к коду? Если ли какая-то ссылка на картинку в самом компьютере? 
</callout>
Нужно открыть свойства изображения (правой кнопкой мыши). Там есть расположение, это и есть ссылка на картинку на компьютере. Мы можем просто скопировать и добавить. Но тогда наша картинка будет привязана к нашему компьютеру, а мы знаем, что сайт это веб-страничка, которая доступна в интернете и картинка там должна храниться всегда. 
<callout icon="👨‍🎓" color="gray_bg">
	Если вы выключите свой компьютер, сможет ли сайт её отображать? 
	<callout icon="🙋‍♂️" color="yellow_bg">
		Нет.
	</callout>
</callout>
<callout icon="💡" color="gray_bg">
	Здесь можно вспомнить, что нужно для того, чтобы наш сайт работал (хостинг и домен).
</callout>
<callout icon="👨‍🎓" color="gray_bg">
	Может ли наш компьютер выступать хостингом? 
</callout>
Да, если он всегда включён. Но это не очень удобно. Если мы хотим, чтобы она хранилась на сайте всегда, мы не сможем просто добавить весь этот адрес в код. 
Нам понадобится загрузить наш файл и картинку на сервер.
Нам нужно на рабочем столе создать папку, например, “Сайт Костя”, внутрь положить файл и нашу картинку. 
В папке “Сайт Костя” нужно создать ещё одну папку, в которой будут храниться наши картинки и назвать её “images”. Нужно добавить нашу картинку в папку “images” и сразу переименовать картинку в понятное нам название, так как мы его тоже будем указывать, например, “Швейцария”. Теперь нужно посмотреть в каком формате наша картинка (это можно посмотреть в свойствах). То есть полное название файла будет “Швейцария.jpg”.
## Ссылка на изображение
У нас есть папка “Сайт”, внутри папки есть файл “index.html”. “index.html” является дочерним файлом для “Сайт Костя”. Ещё внутри есть папка “images” - дочерняя для папки “Сайт Костя”. В папке “image” лежит картинка, которая является дочерней для папки “images”. Внутри файловой системы мы можем ссылаться друг на друга, на объекты. Нам не нужно привязываться к хранению на компьютере. 
Итак, чтобы добавить изображение нам нужно указать название папки, поставить слеш и указать название и тип файла картинки, то есть тег будет выглядеть так:
```html
<img src=”images/Швейцария.jpg”/>
```
Добавьте свою картинку.
# Ссылка
## Тег <span color="blue">\<a\></span>
Сейчас добавим ссылку на кнопку. 
Чтобы добавить кнопку нам понадобится тег <span color="blue">**\<a\>**</span>, который позволяет добавить гиперссылку.
Давайте перед  <span color="blue">**\<button\>**</span> добавим тег <span color="blue">**\<a\>**</span>.  Это парный тег, значит его нужно закрыть. 
## Атрибут href
Для тега <span color="blue">**\<a\>**</span> также нужен атрибут <span color="red">*href*</span>*,* с помощью которого мы и добавим ссылку, на которую сможем перейти.
Наш тег будет выглядеть так 
```html
<a href=”https://ru.wikipedia.org/wiki/Швейцария”></a>
```
Сейчас у нас нет никакого содержимого для тега <span color="blue">**\<a\>**</span>. Давайте добавим какой-то текст между открывающимся и закрывающимся тегом <span color="blue">**\<a\>**</span>:
```html
<a href=”https://ru.wikipedia.org/wiki/Швейцария”>Привет</a>
```
Сохраняем файл и просматриваем наш сайт. Теперь на наш текст можно нажимать и мы перейдём по ссылке.
<callout icon="👨‍🎓" color="gray_bg">
	Но как нам сделать так, чтобы мы переходили на ссылку после нажатия кнопки?
</callout>
Нам нужно поместить кнопку, как содержимое тега <span color="blue">**\<a\>**</span>.
```html
<a href=”https://ru.wikipedia.org/wiki/Швейцария”>
	<button>Подробнее</button>
</a>
```
Весь код должен выглядеть так:
```html
<html>
  <head> </head>
  <body>
    <h1>Швейцария</h1>
    <p>
      Швейцария — одна из самых развитых стран мира, с самым высоким номинальным
      богатством взрослого населения и восьмым по величине ВВП на душу
      населения. Она занимает передовые места по ряду международных показателей,
      включая экономическую конкурентоспособность, инновационность и развитие
      человеческого потенциала.
    </p>
    <p>
      Цюрих, Женева и Базель вошли в десятку лучших городов мира по качеству
      жизни, при этом Цюрих занял второе место.<br />
      Три швейцарских города — Цюрих, Базель и Лозанна — оказались в первой
      сотне научно-технологических кластеров мира.
    </p>
    <a
      href="https://ru.wikipedia.org/wiki/%D0%A8%D0%B2%D0%B5%D0%B9%D1%86%D0%B0%D1%80%D0%B8%D1%8F"
    >
      <button>Подробнее</button>
    </a>
    <img src="images/Швейцария.jpg" />
  </body>
</html>
```
Сохраняем и проверяем на сайте.
В конце нужно сохранить папку с сайтом на ЯндексДиск.
