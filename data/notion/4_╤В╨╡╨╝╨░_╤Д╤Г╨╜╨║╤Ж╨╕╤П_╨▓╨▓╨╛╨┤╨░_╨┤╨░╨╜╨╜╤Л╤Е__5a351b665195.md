# Вывод данных в языке С
Ранее мы уже говорили, что одной из основных задач программирования является обработка информации, поэтому  языки программирования имеют средства для ввода и вывода информации. 
<callout icon="🧑‍🎓" color="gray_bg">
	Каким образом осуществляется ввод данных в языке Python?
	<callout icon="🙋‍♂️" color="yellow_bg">
		Функция input().
	</callout>
</callout>
В языке С за ввод информации отвечает функция <span color="blue">**scanf()**</span>**.**
Функция <span color="blue">**scanf()**</span> используется для считывания данных введенных с клавиатуры в программу. Она может считывать данные всех базовых типов.  автоматически конвертировать их в нужный внутренний формат и записывать в переменные.
## Синтаксис функции
```c
scanf(форматная_строка, аргумент)
```
<span color="blue">**Форматная_строка**</span> нужна для того, чтобы указать вид, в котором информация будет считана. К каждому элементу нужно соответственно указать спецификацию преобразования.
## **Спецификаторы формата**
На прошлом занятии мы уже использовали спецификаторы для вывода значений переменных.
<callout icon="🧑‍🎓" color="gray_bg">
	Какие спецификаторы вы помните и как они записываются?
	<callout icon="🙋‍♂️" color="yellow_bg">
		Перед ними ставится значок процента:<br>%d, %c
	</callout>
</callout>
```c
printf("a = %c \n", a);	// вывод символьного значения переменной a
printf("x = %d \n", x);	// вывод целочисленного значения переменной x
```
Какие могут быть спецификаторы формата:
<span color="blue">**%d**</span> — целое число типа <span color="blue">**int**</span> со знаком в десятичной системе счисления
<span color="blue">**%f**</span><span color="blue"> </span>— вещественный формат (числа с плавающей точкой типа <span color="blue">**float**</span>)
<span color="blue">**%lf**</span><span color="blue"> </span>— вещественный формат двойной точности (числа с плавающей точкой типа <span color="blue">**double**</span>)
<span color="blue">**%c**</span> — символьный формат
```c
scanf("%d", аргументы) // будем принимать целочисленное значение
```
## **Адреса переменных**
<span color="blue">**Аргумент**</span> функции <span color="blue">**scanf()**</span> используются для приема введенных значений. В качестве аргумента передается адрес переменной, которая будет получать введенное значение.
Для получения адреса переменной перед её именем ставится знак амперсанда <span color="blue">**&**</span>. Например, если переменная называется <span color="blue">**age**</span>, то её адрес мы можем получить с помощью выражения <span color="blue">**&age**</span>.
- <span color="blue">**age**</span> - имя переменной
- <span color="blue">**&age**</span> - адрес переменной <span color="blue">**age**</span>
Рассмотрим небольшой пример.
```c
// cчитать целое число и присвоить его значение переменной x

int x;            // объявляем переменную с именем x и типом данных int
scanf("%d", &x);	// обращаемся к функции для считывания данных и записываем введенные данные в переменную x
```
В примере выше:
<span color="blue">**%d**</span>** **- спецификатор формата, который показывает, что мы принимаем целое число типа <span color="blue">**int**</span>
<span color="blue">**&x**</span>** **- адрес переменной <span color="blue">**x**</span>
Таким образом мы сможем ввести данные с клавиатуры, они будут считаны в виде целочисленного значения и записаны в переменную **x**, расположенную по адресу **&x**
<callout icon="💡" color="gray_bg">
	Обратите внимание, что переменная должна быть объявлена перед использованием функции <span color="blue">**scanf()**</span>
</callout>
Сравним с аналогичным действием в Python:
