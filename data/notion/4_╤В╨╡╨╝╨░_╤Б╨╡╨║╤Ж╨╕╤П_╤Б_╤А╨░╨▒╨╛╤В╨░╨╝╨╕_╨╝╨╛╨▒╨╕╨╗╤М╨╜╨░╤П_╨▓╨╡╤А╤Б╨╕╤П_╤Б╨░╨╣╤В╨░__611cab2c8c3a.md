# Вторая секция
## Заголовок
- настроить заголовок исходя из стиля заголовков первой секции
	- цвет
	- размер
	- шрифт
## Фон
- настроить фон исходя из стиля заголовков первой секции
	- цвет
## Разделитель
- добавить разделитель между секциями
# Работы
- создать папку на рабочем столе
- выгрузить в папку работы, сделанные в Adobe Photoshop
- структурировать работы по названиям
# Галерея
- звгрузить работы в галерею
- разобрать основные свойства галереи
	- общие
		- тип
		- колонки
		- отступы
		- ссылки
		- подписи
	- стиль
		- радиус границ
		- тень
- добавить подписи к каждой работе в формате 
`Номер темы - Название темы - Название работы`
> *Пример<br>1 тема - Кисть - Ночь*
# Мобильная версия
- настроить мобильную версию сайта
	- параметры наложения на фон
	- размер текста
	- отступы
# Публикация сайта
- опубликовать сайты
- проверить на телефоне
