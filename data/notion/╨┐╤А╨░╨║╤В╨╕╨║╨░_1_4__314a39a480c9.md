<summary>Задача: Инженерный калькулятор (изучаем \<cmath\>)</summary>
	Мы начнем с довольно простой и базовой задачи. Написать калькулятор
	Условие: 
	Программа должна:
	1. Показать меню действий:
		- **1** — сложение
		- **2** — вычитание
		- **3** — умножение
		- **4** — деление
		- **0** — выход из программы
	2. Спросить у пользователя два числа
	3. Выполнить выбранную операцию (!при делении проверять, что знаменатель не равен нулю!)
	4. Вывести результат 
	5. После выполнения снова показывать меню, пока пользователь не выберет **0**
	Код для решения задачи и ход решения детей может отличаться от предложенного ниже, главное чтобы работало :) Если дети захотят вместо if-else использовать switch-case - пожалуйста! 
	```c++
#include <iostream>
using namespace std;

int main() {
    int choice = -1; // любое число, кроме 0

    while (choice != 0) {
        double a, b;

        cout << "\n=== КАЛЬКУЛЯТОР ===\n";
        cout << "1. Сложение\n";
        cout << "2. Вычитание\n";
        cout << "3. Умножение\n";
        cout << "4. Деление\n";
        cout << "0. Выход\n";
        cout << "Выберите действие: ";
        cin >> choice;

        if (choice == 0) {
            cout << "Выход из программы...\n";
            break;
        }

        cout << "Введите два числа: ";
        cin >> a >> b;

        if (choice == 1) {
            cout << "Результат: " << a + b << endl;
        }
        else if (choice == 2) {
            cout << "Результат: " << a - b << endl;
        }
        else if (choice == 3) {
            cout << "Результат: " << a * b << endl;
        }
        else if (choice == 4) {
            if (b == 0)
                cout << "Ошибка: деление на ноль!" << endl;
            else
                cout << "Результат: " << a / b << endl;
        }
        else {
            cout << "Такой операции нет!" << endl;
        }
    }

    return 0;
}
	```
	После того как дети реализовали обычный калькулятор, мы расширяем функционал, добавив **математические функции** из библиотеки **cmath**.
	Они уже знакомы с циклом, условиями, вводом/выводом — теперь мы усложняем меню.
	## Новое условие
	Программа должна:
	1. Показать расширенное меню действий:
		- **1** — сложение
		- **2** — вычитание
		- **3** — умножение
		- **4** — деление
		- **5** — степень (функция `pow`)
		- **6** — квадратный корень (`sqrt`)
		- **0** — выход из программы
	2. Выполнять операции:
		- для пунктов **1–4** спрашивать **два числа**,
		- для остальных — **одно число** (кроме pow → 2 числа).
	3. Проверять ошибки (деление на ноль, sqrt от отрицательного).
	4. После выполнения снова показывать меню, пока пользователь не выберет **0**
	```c++
#include <iostream>
#include <cmath>         
using namespace std;

int main() 
    int choice;
    do {
        cout << "=== ИНЖЕНЕРНЫЙ КАЛЬКУЛЯТОР ===\n\n";
        cout << "1.  Сложение\n";
        cout << "2.  Вычитание\n";
        cout << "3.  Умножение\n";
        cout << "4.  Деление\n";
        cout << "5.  x^y\n";
        cout << "6.  √x\n";
        cout << "0.  Выход\n\n";
        cout << "Выберите операцию: ";
        cin >> choice;

        if (choice == 0) break;

        double a, b = 0;

        if (choice >= 1 && choice <= 5) {
            cout << "Введите два числа: ";
            cin >> a >> b;
        } else if (choice == 6) {
            cout << "Введите число: ";
            cin >> a;
        }

        switch (choice) {
            case 1:  cout << "Результат: " << a + b << endl; break;
            case 2:  cout << "Результат: " << a - b << endl; break;
            case 3:  cout << "Результат: " << a * b << endl; break;
            case 4:
                if (b == 0) cout << "Ошибка: деление на ноль! \n";
                else cout << "Результат: " << a / b << endl;
                break;
            case 5:  cout << "Результат: " << pow(a, b) << endl; break;
            case 6:
                if (a < 0) cout << "Ошибка: корень из отрицательного!\n";
                else cout << "Результат: " << sqrt(a) << endl;
                break;
            default: cout << "Такой операции нет!\n";
        }

        cout << "\nНажмите Enter для продолжения...";

    } while (true);

    cout << "До встречи!\n";
}
	```

<summary>Задача: Угадай число (изучаем \<ctime\>)</summary>
	Написать игру, в которой компьютер загадывает случайное число от **1 до 100**, а игрок должен его угадать.
	### Требования:
	1. Компьютер загадывает случайное число в диапазоне **1–100**.
	2. У игрока есть **7 попыток**, чтобы угадать число.
	3. После каждой попытки программа сообщает:
		- “Моё число больше”, если загаданное число больше введённого.
		- “Моё число меньше”, если загаданное число меньше введённого.
	4. Если игрок угадал число:
		- программа сообщает, за сколько попыток это произошло;
		- если это меньше, чем лучший результат, устанавливается **новый рекорд**.
	5. Если попытки закончились:
		- программа сообщает правильное число.
	6. После окончания игры программа спрашивает:
		**“Сыграем ещё? (да/нет)”**
	7. Если пользователь отвечает “да”, игра начинается заново с новым числом.
	8. После выхода программа выводит:
		**“Самый лучший результат: X попыток”**,
		где *X* — наименьшее количество попыток, за которое игрок когда-либо угадывал число.
	```c++
#include <iostream>
#include <cstdlib>
#include <ctime>
using namespace std;

int main() {
    srand(time(NULL));
    int record = 999;  // лучший резул ьтат

    string play_again;
    do {
        int secret = rand() % 100 + 1;
        int attempt = 0;
        int guess;
        
        cout << "Я загадал число от 1 до 100!\n";

        do {
            attempt++;
            cout << "Попытка №" << attempt << ": ";
            cin >> guess;

            if (guess < secret) cout << "Моё число больше!\n";
            else if (guess > secret) cout << "Моё число меньше!\n";
            else {
                cout << "УГАДАЛ за " << attempt << " попыток!\n";
                if (attempt < record) {
                    record = attempt;
                    cout << "НОВЫЙ РЕКОРД!!!\n";
                }
                break;
            }

            if (attempt == 7) {
                cout << "Попытки кончились! Было число " << secret << endl;
                break;
            }
        } while (true);

        cout << "Сыграем ещё? (да/нет): ";
        cin >> play_again;

    } while (play_again == "да" || play_again == "Да");

    cout << "Самый лучший результат: " << record << " попыток!\n";
}
	```

<summary>Задача: Числа Армстронга </summary>
	**Числа Армстронга** (названные в честь Майкла Ф. Армстронга<br>(Michael F. Armstrong)), их также называют самовлюбленными числами и<br>совершенными цифровыми инвариантами, – это числа, равные сумме своих<br>цифр, возведенных в степень, равную количеству цифр. Например, наименьшее<br>число Армстронга – 153, которое равно 1\^3 + 5\^3 + 3\^3 . Напишите программу,<br>которая выводит все числа армстронга меньше 10\^6
	```c++
#include <iostream>

using namespace std;

int main() {
    for (int num = 1; num < 1000000; ++num) {
        int temp = num;
        int digit_count = 0;
        
        // Подсчет количества цифр
        while (temp > 0) {
            ++digit_count;
            temp /= 10;
        }
        
        temp = num;
        int sum = 0;
        
        // Вычисление суммы цифр в степени digit_count
        while (temp > 0) {
            int digit = temp % 10;
            int power = 1;
            
            // Возведение в степень с помощью цикла
            for (int i = 0; i < digit_count; ++i) {
                power *= digit;
            }
            
            sum += power;
            temp /= 10;
        }
        
        if (sum == num) {
            cout << num << endl;
        }
    }
    
    return 0;
}
	```

<summary>Задача: Куб суммы его цифр</summary>
	Найти двухзначное число, обладающее тем свойством, что куб суммы его цифр равен квадрату самого числа.
	```c++
#include <iostream>

using namespace std;

int main() {
    for (int n = 10; n < 100; n++) {
        int a = n / 10;  // Первая цифра (десятки)
        int b = n % 10;  // Вторая цифра (единицы)
        int s = a + b;   // Сумма цифр
        
        if (s * s * s == n * n) {  // Проверка: куб суммы == квадрат числа
            cout << n << endl;
        }
    }
    return 0;
}
	```
