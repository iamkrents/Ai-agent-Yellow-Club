# Условия задачи
<callout icon="✅" color="gray_bg">
	Изучить условия задачи и создать проект.
</callout>
Создать приложение “Камень - ножницы - бумага”, которое рассчитано на 2 игроков.
**Дизайн**
- cверху и снизу расположены кнопки с текстом “Играть”
	- кнопки и текст повернуты к игроку
	- цвет кнопок - желтый
	- цвет текста - черный.
- под/над кнопками располагаются надписи с пустым текстом
- после надписей - серые области
	- сделаны с помощью растянутого TextView
	- изначально области пустые. 
- посередине располагается надпись “Результат”
**Функционал**
При нажатии на кнопку играть в серых областях появляется рандомно выбранный элемент (камень-ножницы-бумага). После нажатия на надпись “Результат” у каждого игрока над/под кнопками появятся надписи о том, выиграли они или проиграли.

# Логика приложения
<callout icon="✅" color="gray_bg">
	Прописать логику работы приложения.
</callout>
- объявляем переменные
- создаем обработчик событий 
- создаем функцию генерация рандомного элемента из списка
```kotlin
package com.yxl.calculator

import android.os.Bundle
import android.widget.TextView
import androidx.activity.enableEdgeToEdge
import androidx.appcompat.app.AppCompatActivity
import androidx.core.view.ViewCompat
import androidx.core.view.WindowInsetsCompat
import kotlin.random.Random

class MainActivity : AppCompatActivity() {

    // Список возможных элементов
    val list = listOf("камень", "бумага", "ножницы")

    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        enableEdgeToEdge()
        setContentView(R.layout.activity_main)
        // Создаем переменные для элементов верстки
        val tvResult = findViewById<TextView>(R.id.tvResult)
        val tvTop = findViewById<TextView>(R.id.tvTopText)
        val tvTopResult = findViewById<TextView>(R.id.tvTopResult)
        val tvBottom = findViewById<TextView>(R.id.tvBottomText)
        val tvBottomResult = findViewById<TextView>(R.id.tvBottomResult)
        val bTop = findViewById<Button>(R.id.bTopPlay)
        val bBottom = findViewById<Button>(R.id.bBottomPlay)
        // Создаем переменные для выбора игроков, по умолчанию они ничего не выбрали
        var pOne:String = ""
        var pTwo:String = ""
        // Переменная которая будет хранить номер игрока, который победил, 1 - сверху, 2 - снизу, 0 - ничья
        // По умолчанию ничья
        var result: Int = 0

        // Не принимаем во внимание, как и раньше
        ViewCompat.setOnApplyWindowInsetsListener(findViewById(R.id.main)) { v, insets ->
            val systemBars = insets.getInsets(WindowInsetsCompat.Type.systemBars())
            v.setPadding(systemBars.left, systemBars.top, systemBars.right, systemBars.bottom)
            insets
        }
        // Обработчик нажатия по кнопке играть у игрока 1, генерируем рандом элемент и ставим в серую область
        bTop.setOnClickListener(){
            pOne = generateRandomValue(list)
            tvTop.text = pOne
            // Очищаем текст результата каждый раз при нажатии на кнопку, для того, чтобы можно было играть много раз
            tvTopResult.text = ""
        }
        // Обработчик нажатия по кнопке играть у игрока 2, генерируем рандом элемент и ставим в серую область
        bBottom.setOnClickListener(){
            pTwo = generateRandomValue(list)
            tvBottom.text = pTwo
            // Очищаем текст результата каждый раз при нажатии на кнопку, для того, чтобы можно было играть много раз
            tvBottomResult.text = ""
        }
        // Обработчик нажатия по надписи Результат, вызываем функцию сравнения элементов
        tvResult.setOnClickListener(){
            result = compareElements(pOne, pTwo)

            // В зависимости от числа, которое нам выдало сравнение, пишем в надписях под/над кнопками Ты выиграл или ты проиграл
            when(result){
                1 -> {
                    tvTopResult.text = "Ты выиграл!!"
                    tvBottomResult.text = "Ты проиграл(("
                }
                2 -> {
                    tvTopResult.text = "Ты проиграл(("
                    tvBottomResult.text = "Ты выиграл!!"
                }
                else -> {
                    tvTopResult.text = "Ничья"
                    tvBottomResult.text = "Ничья"
                }
            }
        }
    }

    // Функция генерации рандомного элемента из списка. Передаем список, на его основе ставим промежутки для индекса
    // от 0 до его размера не включая
    // После чего возвращаем элемент с рандомным индексом
    fun generateRandomValue(list: List<String>): String {
        return list[Random.nextInt(0, list.size)]
    }
    // Функция сравнения элементов, такую писали еще раньше, должно быть все в порядке
    fun compareElements(f: String, s: String): Int {

        if ((f == "камень" && s == "ножницы")
            || (f == "ножницы" && s == "бумага")
            || (f == "бумага" && s == "камень")
        ) {
            return 1
        } else if ((s == "камень" && f == "ножницы")
            || (s == "ножницы" && f == "бумага")
            || (s == "бумага" && f == "камень")
        ) {
            return 2
        } else {
            return 0
        }
    }
}
```
