// Список продуктов

// Задача
// Вы собираетесь в магазин и решили составить список продуктов
// Напишите алгоритм, который:

// Методы списков

// add(element)
// Мы забыли внести 1 продукт, нужно добавить его в конец списка и вывести список в консоль
 println("Введите название продукта: ")
 val newProduct = readln()
 productList.add(newProduct)
// Это сделать не получится поэтому рассказываем о том, что:
/*
	Списки в Котлин по умолчанию не изменяемые, то есть мы вписываем сразу список элементов и не сможем менять этот список
	Для того, чтобы создавать списки, которые можно менять, нужно использовать mutableListOf() или mutableListOf<Type>()
	(мьютабл лист) - изменяемый список. Так что переделаем наш пример в такой вид: 
	val productList = readln().split(" ").toMutableList()
	Так как по умолчанию списки неизменяемые, то и список, полученный в результате split будет тоже неизменяемым, поэтому мы приводим
	его к типу изменяемого списка
	val newProduct = readln()
	productList.add(newProduct)
*/
 println(productList)

// remove(element)
// Мы купили 1 продукт, который нам не нужен и должны внести это изменение в список и вывести новый список в консоль
 val buyedProduct = readln()
 productList.remove(buyedProduct)
 print(productList)

// removeAt(index)
// в функции removeAt() мы вводим индекс элемента, который хотим удалить
// 3-й продукт отсутствует в магазине
// Удалить отсутствующий продукт из списка (с помощью индекса)
// Но оставить его наименование в памяти (удалить из списка, сохранив в отдельную переменную)
// И вывести наименование удаленного продукта в консоль
// Вывести обновленный список
 val missingProduct = productList.removeAt(2)
 println(missingProduct)
 println(productList)

// add(index, element)
// та же функция add() может принимать и другие аргументы, сейчас, чтобы вставить элемент в начало списка, мы
// должны передать индекс, на который мы хотим поставить новый элемент и сам элемент
// Выяснилось, что нам нужен еще один продукт
// Внести этот продукт первым в списке
// Вывести список
 println("Какой продукт срочно понадобился: ")
 val mainProduct = readln()
 productList.add(0, mainProduct)
 print(productList)

// indexOf(element)
// Один из продуктов перестал быть нужен
// Узнать какой и вывести его индекс в консоль
 println("Введите наименование продукта, который перестал быть нужен: ")
 val delProductName = readln()
 val delProductIndex = productList.indexOf(delProductName)
 println(delProductIndex)
// Удалить продукт из списка
 productList.removeAt(delProductIndex)
 println(productList)

// clear()
// Мы решили создать новый список и скопировать туда уже имеющийся (метод copy() или срез)
// Старый список удалим и выведем оба в консоль
 val newProductList = productList.toMutableList() // привести к списку нужно, чтобы newProductList был отличным от productList
// если же этого не сделать, то они будут ссылаться на один и тот же объект постоянно, то есть на productList, можно показать
// что будет если не приводить к списку (убрать toMutableList())
 productList.clear()
 println(newProductList)
