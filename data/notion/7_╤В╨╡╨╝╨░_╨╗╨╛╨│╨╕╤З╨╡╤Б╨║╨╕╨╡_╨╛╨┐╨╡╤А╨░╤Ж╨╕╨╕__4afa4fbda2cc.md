# Логические операции
<callout icon="🧑‍🎓" color="gray_bg">
	Для чего используются логические операции?
	<callout icon="🙋‍♂️" color="yellow_bg">
		Когда нужно совместить выполнение нескольких условий.
	</callout>
</callout>
Логические операции нужны для проверки нескольких условий одновременно. Результатом выполнения логических операций являются 1 или 0, где:
- 1 — выражение истинно
- 0 — выражение ложно
## Логические операторы
В языке С существуют следующие логические операторы:
<span color="blue">**&&**</span> — логическое «И» ( конъюнкция)
<span color="blue">**\|\|**</span> — логическое «ИЛИ» (дизъюнкция)
<span color="blue">**!**</span> — логическое «НЕ» (отрицание)
## Виды логических операций
Синтаксис операций сравнения выглядит так:
```c
условие_1 логический_оператор условие_2
```
Для примера объявим переменные **a**, **b**,** c**, которым присвоим целочисленные значения и проведем с ними логические операции.
```c
#include <stdio.h>

int main()
{
	int a = 9;
	int b = 13;
	int с = 17;

	return 0;
}
```
### **Логическое «И» (конъюнкция)**
```c
условие_1 && условие_2
```
Возвращает 1, если оба условия истинны. Возвращает 0, если хотя бы одно условие ложно.
```c
#include <stdio.h>

int main()
{
	int a = 9;
	int b = 13;
	int c = 17;
	
	a < c;		                        // результатом будет 1, то есть истина, так как 9 меньше 13
	b < c; 		                        // результатом будет 1, то есть истина, так как 13 меньше 17
	a < c && b < c;		                // результатом будет 1, так как оба условия истинны
	printf("%d \n", a < c && b < c); 	// результатом будет вывод в консоль 1

	return 0;
}
```
### **Логическое «ИЛИ» (дизъюнкция)**
```c
условие_1 || условие_2
```
Возвращает 1, если хотя бы одно из условий истинно. Возвращает 0, если оба условия ложны.
```c
#include <stdio.h>

int main()
{
	int a = 9;
	int b = 13;
	int c = 17;
	
	a > c;		                        // результатом будет 0, то есть ложь, так как 9 меньше 17
	b < c; 		                        // результатом будет 1, то есть истина, так как 13 меньше 17
	a > c || b < c;		                // результатом будет 1, так как одно из условий истинно
	printf("%d \n", a > c || b < c); 	// результатом будет вывод в консоль 1

	return 0;
}
```
### **Логическое «НЕ» (отрицание)**
Этот оператор отрицает выражение, к которому он был применён. Если условие было истинным, оно становится ложным, а если было ложным, то становится истинным.
```c
#include <stdio.h>

int main()
{
	int a = 9;
	int b = 13;
	int c = 17;
	
	a > c;		  // результатом будет 0, то есть ложь, так как 9 меньше 17
	!(a > c);		// результатом будет 1, то есть истина, так как условие в скобках ложно, но оператор логического отрицания возвращает обратный результат
	
	printf("%d \n", !(a > c)); 	      // результатом будет вывод в консоль 1
	
	return 0;
}
```
В этом примере после добавления знака <span color="blue">**!**</span>** **выражение, которое до этого было ложным, стало истинным.
