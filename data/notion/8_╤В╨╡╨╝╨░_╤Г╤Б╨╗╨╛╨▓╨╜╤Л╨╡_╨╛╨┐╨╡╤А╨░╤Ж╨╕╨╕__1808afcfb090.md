# **Условные операции**
Условные операции — это выражения, которые проверяют, соответствуют ли данные условию, и выполняют действие в зависимости от результата.
> *Например, выражение:*
	*«Если на улице идёт дождь, то нужно взять зонт»*
	*Если представить этот пример как программу, то у нас есть:*
	- *условие: «если на улице идет дождь»*
	- *действие, которое запускается при выполнении условия: «нужно взять зонт»*
## Условные операторы
В условных операциях используются операторы, позволяющие управлять ходом выполнения программы в зависимости от выполнения определённых условий:
- <span color="blue">**if **</span>
- <span color="blue">**else**</span>
- <span color="blue">**switch**</span>
- <span color="blue">**case**</span>
Эти операторы формируют условные конструкции.
## Виды условных конструкций
В языке С существует следующие условные конструкции:
- <span color="blue">**if **</span>- позволяет проверить условие и выполнить один блок кода, если условие истинно
- <span color="blue">**if-else**</span>** **- позволяет проверить условие и выполнить один блок кода, если условие истинно, и другой блок кода, если условие ложно
- <span color="blue">**switch-case**</span> - используется для выбора одного из нескольких блоков кода в зависимости от значения переменной
### **Конструкция **<span color="blue">**if**</span>
Оператор** **<span color="blue">**if**</span> проверяет истинность условия, и если оно истинно, выполняет блок инструкций.
Синтаксис конструкции <span color="blue">**if**</span>:
```c
if (условие) {
	инструкция;
}
```
<callout icon="💡" color="gray_bg">
	Обратите внимание, что перед инструкцией обязательно ставится отступ, иначе инструкция не будет выполняться после проверки условия, даже если условие было истинным.
</callout>
Рассмотрим пример.
```c
#include <stdio.h>

int main()
{
	int a = 15;                     // объявляем переменную и присваиваем ей значение
	
	if (a > 10) {                   // вводим проверку условия “если a > 10, то”
		printf("a > 10");             // выводим в консоль "a > 10" при соблюдении условия
	}
	
	return 0;
}
```
Вычисляется значение условного выражения. Если получается истина, то выполняется инструкция, а если ложь, то  алгоритм завершается.
### **Конструкция **<span color="blue">**if-else**</span>
Если мы хотим проверить условие и в случае, если оно ложно, выполнить другой блок код, необходимо использовать конструкцию <span color="blue">**if-else**</span>.
Синтаксис конструкции <span color="blue">**if-else**</span>:
```c
if (условие) {
	инструкция_1;
}
else {
	инструкция_2;
}
```
Продолжим предыдущий пример, но заменим значение в переменной.
```c
#include <stdio.h>

int main()
{
	int a = 7;                         // объявляем переменную и присваиваем ей значение
	
	if (a > 10) {                      // вводим проверку условия “если a > 10, то”
		printf("a > 10");               
	}
	else {                             // запускаем блок кода, если условие не соблюдено
		printf("a < 10");                // выводим в консоль "a < 10" при несоблюдении условия
	}
	
	return 0;
}
```
Эта программа выведет в консоль сообщение **a \< 10**, так как выражение условия **a** **\> 10 **ложно.

# Дополнительный материал
### **Конструкция **<span color="blue">**switch-case**</span>
Эта конструкция используется для выбора одного из нескольких блоков кода в зависимости от значения переменной
Синтаксис этой конструкции выглядит следующим образом:
```c
switch (выражение) {
case константа_1:
	инструкция_1;
	break;
case константа_2:
	инструкция_2;
	break;
case константа_3:
	инструкция_3;
	break;
default:
	инструкция_4;
}
```
Как работает эта конструкция:
- cначала вычисляется значение выражения в скобках
- затем оно сравнивается со всеми константами (значения констант должны отличаться друг от друга)
- когда обнаруживается совпадение, инструкции, ассоциированные с соответствующим <span color="blue">**case**</span>, выполняются до тех пор, пока не встретится оператор <span color="blue">**break**</span>
- если ни одна из констант не совпадает со значением, тогда выполняются инструкции после оператора <span color="blue">**default**</span>
- если <span color="blue">**default**</span> отсутствует и ни одна из констант не подошла, то алгоритм завершается
Например, в следующей программе происходит проверка, равно ли целое число одному из 3-х значений.
```c
#include <stdio.h>

int main()
{
	int a = 5;                         // объявляем переменную и присваиваем ей значение
	
	switch (a) {
	case 3: 			                     // проверяем равно ли значение переменной числу 3
		printf("a = 3");	               // выводим сообщение при совпадении
		break; 			                     // выходим из ветки switch
	case 4: 		
		printf("a = 4");
		break;
	case 5: 		
		printf("a = 5");
		break;
	default: 				                   // если нет совпадений
		printf("a < 3 или a > 5");	     // выводим это сообщение
	}
	
	return 0;
}
```
