# **Переменные**
<callout icon="🧑‍🎓" color="gray_bg">
	Что такое переменные?
	<callout icon="🙋‍♂️" color="yellow_bg">

Для хранения данных в языках программирования используются переменные. Чтобы к ним было удобно обращаться, переменным присваиваются имена. Это могут быть короткие имена, например, <span color="blue">**x**</span> и <span color="blue">**y**</span>. Или более описательные - <span color="blue">**age**</span>, <span color="blue">**sum**</span>, <span color="blue">**name**</span>.

	</callout>
</callout>
Супер, все верно.
<callout icon="🧑‍🎓" color="gray_bg">
	А какие правила для присвоения имен переменным есть?
	<callout icon="🙋‍♂️" color="yellow_bg">
		- латиница<br>- не начинается с чисел<br>- нельзя использовать символы
	</callout>
</callout>
Правила присвоения имён переменным в языке С и Python аналогичны:
- должны быть записаны латиницей
- могут содержать буквы, цифры и подчёркивания
- должны начинаться с буквы или символа подчёркивания
- не могут содержать пробелов или специальных символов, таких как !, #, %
- чувствительны к регистру (Name и name — разные переменные)
- зарезервированные слова (например, int, float, auto, break, char, const и др.) не могут использоваться в качестве имён
- в коде не может быть двух переменных с полностью идентичными именами
<callout icon="💡" color="gray_bg">
	Переменная должна быть объявлена раньше, чем она будет использоваться.
</callout>

# **Типы данных**
Типы данных определяют, какие операции могут быть выполнены над данными и сколько памяти компьютера требуется для хранения этих данных. В языке С определены следующие базовые типы данных:
## **Целые числа - **<span color="blue">**int**</span>
Числа без десятичной части, такие как 1, 2, -100 и т.д. Обычно занимают от 4 до 8 байт памяти. Если брать основные платформы - 64-разрядные Windows, Linux (вместе с Android) и MacOS, то размер <span color="blue">**int**</span> составляет 4 байта. Диапазон предельных значений на Windows  −2 147 483 648 до 2 147 483 647 (при 4 байтах) и выше. 
💡Есть и другие типы данных озночающие целое число: **long, unsigned int, unsigned long и.т.д (Инфа на будущее)**
<empty-block/>
```c
int x;		// объявление переменной с именем “x” и типом данных “int”
x = 5;		// присвоение переменной “x” значения “3”
```
## **Вещественные числа -**<span color="blue">** float**</span>** и **<span color="blue">**double**</span>
Числа с десятичной частью, такие как 3.14, -0.5 и т. д. Занимают от 4 до 12 байт памяти в зависимости от точности и диапазона.
```c
float y;	     // объявление переменной с именем “y”  и типом данных “float”
y = 3.2;	     // присвоение переменной “y” значения “3.2”

double z;	     // объявление переменной с именем “z”  и типом данных “double”
z = 3.2148;	   // присвоение переменной “z” значения “3.2148”
```
Разница между <span color="blue">**float**</span> и <span color="blue">**double**</span> заключается в том, что double в два раза более детализирован, чем <span color="blue">**float**</span>: в нем может быть вдвое больше чисел после запятой.
## **Символьные строки - **<span color="blue">**char**</span>
Cимволы, такие как буквы, цифры или специальные символы. Например: ‘a’, ’1’, ‘!’. Обычно занимают 1 байт на символ плюс дополнительные байты для хранения длины строки.
```c
char letter;	   // объявление переменной с именем “letter“  и типом данных “char”
letter = 'Y';	   // присвоение переменной “letter” значения “Y”
```
<callout icon="💡" color="gray_bg">
	При присвоении значений переменным важно следить за наличием/отсутствием кавычек. Например, при работе с переменной с **символьным типом** данных char, значения помещаются в **одинарные кавычки**. А при работе с **числовыми типами** данных <span color="blue">**int**</span>, <span color="blue">**float**</span>, <span color="blue">**double**</span> присваиваемые  переменным числа записываются **без кавычек**.
</callout>

## **Спецификаторы формата**
Нужны для того, чтобы указывать вид, в котором информация будет выведена.
- <span color="blue">**%c**</span> — символьный формат
- <span color="blue">**%d**</span><span color="blue"> </span>— целое число типа <span color="blue">**int**</span> со знаком в десятичной системе счисления
- <span color="blue">**%f**</span><span color="blue"> </span>— вещественный формат (числа с плавающей точкой типа <span color="blue">**float**</span>)
- <span color="blue">**%lf**</span><span color="blue"> </span>— вещественный формат двойной точности (числа с плавающей точкой типа <span color="blue">**double**</span>)
<callout icon="💡" color="gray_bg">
	Отличительной чертой спецификатора является наличие символа <span color="blue">**%**</span> в начале.
</callout>
То есть, чтобы вывести данные, хранящиеся в переменной, недостаточно указать ее название. Необходимо также указать спецификатор формата. Синтаксис вывода значений переменных в языке C выглядит так:
```c
printf(”спецификатор_формата”, имя_переменной);
```
Рассмотрим на примере как это работает:
```c
// Объявить переменную и присвоить ей целочисленное значение
// Вывести значение в консоль

int x = 3;
printf("%d", x);   // иснользуется спецификатор формата %d, так как переменная хранит целочисленное значение
```

## **Управляющие символы**
<callout icon="🧑‍🎓" color="gray_bg">
	Какой результат вывелся в консоль?
	<callout icon="🙋‍♂️" color="yellow_bg">
		Все в одну строку.
	</callout>
</callout>
Для того, чтобы вывести  значение каждой переменной с новой строки необходимо использовать управляющие символы. Именно они управляют расположением выводимых символов. 
Основные управляющие символы:
- <span color="blue">**\\n**</span> — перевод на новую строку;
- <span color="blue">**\\b**</span> — возврат на символ;
- <span color="blue">**\\r**</span><span color="blue"> </span>— возврат на начало строки.
<callout icon="💡" color="gray_bg">
	Отличительной чертой управляющего символа является наличие символа <span color="blue">**\\**</span> в начале.
</callout>
Синтаксис выглядит так:
```c
printf(”спецификатор_формата управляющий_симвом”, имя_переменной);
```
Рассмотрим на примере как это работает:
```c
// Объявить 2 переменные и присвоить им целочисленные значения
// Вывести значения в консоль в 2 строки

int x = 3;
int y = 5;

printf("%d \n", x);   // иснользуется управляющий символ \n для перехода на новую строку
printf("%d \n", y);
```

## Одновременный вывод строки(текста) и значения переменной
Чтобы совместить вывод текста (строки) и значения переменной, нужно добавить текст внутрь кавычек перед спецификатором формата.
Синтаксис выглядит так:
```c
printf(”текст спецификатор_формата управляющий_симвом”, имя_переменной);
```
Рассмотрим на примере как это работает:
```c
// Объявить переменную и присвоить ей целочисленное значение
// Вывести значение в консоль с дополнительным текстом

int x = 3;
printf("Значение x равно %d \n", x);   // добавили текст перед спецификатором
```
Здесь <span color="blue">**"Значение x равно %d \\n" **</span>- это** ** строка форматов, которая состоит из следующих элементов:
- текста, представленного для непосредственного вывода - <span color="blue">**Значение x равно**</span>
- спецификатора формата, предназначенных для вывода значений переменных различных типов - <span color="blue">**%d**</span>
- управляющего символа - <span color="blue">**\\n**</span>
И <span color="blue">**x**</span> после запятой - это переменная.
